# How I built a AI Agent server

**Author:** Yash Kumar Saini
**Published:** April 20, 2025

## Overview
Describes building "Hack Agno," an AI agent deployed as an API service using Python and FastAPI with the Agno framework. Agno is a lightweight library for building Agents with memory, knowledge, tools and reasoning, enabling Reasoning Agents, Multimodal Agents, Teams of Agents, and Agentic Workflows.

## Key Concepts
- Two AI agents communicate with a FastAPI backend
- Different API routes powered by specialized agents
- Retro terminal-like UI for the frontend
- Suggestion from community: use a FastAPI library that converts endpoints to MCP for LLM tool integration
