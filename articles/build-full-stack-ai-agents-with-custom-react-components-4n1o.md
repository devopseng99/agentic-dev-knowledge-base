# Build Full-Stack AI Agents with Custom React Components (CopilotKit + CrewAI)
**Author:** Nathan Tarbert
**Published:** March 28, 2025

## Overview
Demonstrates integrating CrewAI agents into applications using CopilotKit, enabling agents to take control of applications, communicate actions, and generate completely custom UI through a new category of Agent-Native applications.

## Key Concepts

### Agent-Native Applications
A new category including Replit's AI Agent, Vercel's v0, and ChatGPT's Canvas that deeply integrate AI agents into native applications, enabling deep human/AI collaboration and human-in-the-loop flows.

### CopilotKit + CrewAI Integration
- CoAgents intro with CrewAI
- CLI tool for adding Crews to React apps
- Shared agent state implementation
- React component for agent streaming
- Agentic Generative UI
