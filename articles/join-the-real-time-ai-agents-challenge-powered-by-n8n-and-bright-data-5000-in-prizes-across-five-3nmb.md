# Join the Real-Time AI Agents Challenge powered by n8n and Bright Data
**Author:** Jess Lee
**Published:** August 13, 2025

## Overview
Developer challenge to build AI agents using n8n automation and Bright Data web data infrastructure. $5,000 in prizes across five winners, running through August 31, 2025.

## Key Concepts
- n8n's AI Agent node with Bright Data's verified node required
- Public n8n chat interface or demo video mandatory
- n8n workflow JSON in GitHub Gist or repository
- $250 Bright Data credits for participants
- Five winners each receive $1,000 USD + DEV++ Membership
- Data not intended for reuse, resale, or redistribution
