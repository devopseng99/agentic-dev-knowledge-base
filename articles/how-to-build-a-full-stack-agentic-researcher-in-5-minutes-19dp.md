# How to Build a Full Stack Agentic Researcher in 5 Minutes
**Author:** Nathan Tarbert
**Published:** February 26, 2025

## Overview
Introduction to CopilotKit's CoAgents for building Agent-Native applications with embedded chat interfaces that actively take actions within app frontend functions. Covers shared state, streaming, generative UI, and human-in-the-loop patterns.
