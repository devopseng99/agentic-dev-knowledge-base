# Getting Started with Kafka Connect
**Author:** Ankit Sood
**Published:** 2025

## Overview
Comprehensive guide to Kafka Connect for streaming data between Kafka and other systems, focusing on open-source setup rather than Confluent Cloud.

## Key Concepts
- Source connectors for data ingestion into Kafka
- Sink connectors for data export from Kafka
- Standalone vs distributed mode deployment
- Single Message Transforms (SMTs) for data manipulation
- Connector configuration and management via REST API
- Essential for building AI data pipelines that connect disparate systems
