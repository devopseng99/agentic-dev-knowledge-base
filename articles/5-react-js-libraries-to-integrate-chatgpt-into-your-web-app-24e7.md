# 5 React / JS Libraries To Integrate ChatGPT Into Your Web App
**Author:** Salmen Hichri
**Published:** December 15, 2023

## Overview
Five open-source libraries for embedding AI chatbots into React/JS web applications.

## Key Concepts
1. **NLUX**: React components with adapter connections to ChatGPT and LLAMA2
2. **Botanic**: Full-stack serverless React framework for multi-platform chatbots
3. **Bottender**: Framework built on Messaging APIs with request batching
4. **React Chatbot Kit**: Frontend-focused library with message parsers and action providers
5. **OpenAI JS SDK**: Official JavaScript library for ChatGPT, designed for Node.js + React
