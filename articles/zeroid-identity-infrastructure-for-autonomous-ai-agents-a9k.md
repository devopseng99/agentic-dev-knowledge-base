# ZeroID: Identity Infrastructure for Autonomous AI Agents
**Author:** Kunal Kumar
**Published:** April 16, 2026

## Overview
ZeroID is an open-source identity layer for AI agents providing cryptographically verifiable identities, agent-to-agent delegation, and real-time revocation.

## Key Concepts
- Individual cryptographically verifiable identities for each agent
- Agent-to-agent delegation capabilities
- Real-time revocation functionality
- Scoped and downscoped credentials for sub-agents
- Auditable delegation chains

### Technical Foundation
Built upon OAuth 2.1, RFC 8693 token exchange, and WIMSE/SPIFFE-style identities.

**Repository:** https://github.com/highflame-ai/zeroid
