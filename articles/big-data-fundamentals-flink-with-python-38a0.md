# Big Data Fundamentals: Flink with Python
**Author:** DevOps Fundamentals  **Published:** 2024

## Overview
PyFlink tutorial demonstrating Apache Flink stream processing with Python. Covers machine learning prediction use cases within Flink jobs, including loading ML models inside Python UDFs as a dependency-management pattern. Part of the Big Data Fundamentals series.

## Key Concepts
- PyFlink: Python API for Apache Flink stream and batch processing
- Loading machine learning models inside Python UDFs
- Dependency management for ML models in Flink Python jobs
- Scenarios: ML prediction, sentiment analysis, recommendation, classification on streaming data
- Python UDF integration for Flink ML feature engineering

Use cases enabled by PyFlink + ML:
- Real-time sentiment analysis on streaming text
- Recommendation systems with online feature computation
- Classification tasks on IoT sensor streams
- Online inference / feature engineering runtime patterns
