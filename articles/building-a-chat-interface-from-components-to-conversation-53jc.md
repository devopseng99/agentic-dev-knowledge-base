# Building a Chat Interface: From Components to Conversation
**Author:** Sebastiao Gazolla Jr
**Published:** September 22, 2025

## Overview
Java-based chat interface integrating MCPService, LLMClient, and SimpleInference into a functional MCP-powered chat application with natural language tool interactions.

## Key Concepts
- ChatInterface class for natural language MCP tool interaction
- Weather queries, file operations via conversation
- Environment-based API key configuration
- Multi-MCP server integration
- Single-user, basic context retention (educational design)
