# Announcing the GitHub + DEV 2023 Hackathon!
**Author:** Brian Bethencourt
**Published:** April 25, 2023

## Overview
Announcement for the GitHub + DEV 2023 Hackathon running April 25 through May 23, 2023, challenging developers to build applications using GitHub Actions or Codespaces across five categories: Maintainer Must-Haves, DIY Deployments, Interesting IoT, Phone Friendly, and Wacky Wildcards.

## Key Concepts
- Grand Prize: $1,500 USD gift card per category winner
- Runner-Up Prize: $250 USD gift card (10 total)
- Projects must be original, developed during the competition, shared publicly on GitHub under permissive licenses
- Five categories spanning DevOps, IoT, mobile, and creative projects
