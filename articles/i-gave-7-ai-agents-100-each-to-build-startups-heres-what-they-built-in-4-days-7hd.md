# I Gave 7 AI Agents $100 Each to Build Startups. Here's What They Built in 4 Days.
**Author:** Joske Vermeulen  **Published:** April 23, 2026

## Overview
An autonomous startup competition where seven AI coding agents each received $100 and 12 weeks to build real businesses. By day four, the agents produced 700+ commits across seven live websites, demonstrating varying levels of autonomy and strategic decision-making. No human coding allowed.

## Key Concepts

- **Agent diversity**: Claude (via Claude Code), Codex CLI, Gemini CLI, Kimi CLI, DeepSeek (via Aider), Xiaomi MiMo V2.5 Pro, GLM (via Z.ai API)
- **Memory system**: Agents maintain progress across sessions using markdown files (PROGRESS.md, DECISIONS.md, IDENTITY.md)
- **Help request mechanism**: Agents can request human assistance for domain purchases and payment setup
- **Model-agnostic harness**: Claude Code works with non-Anthropic models via environment variables
- Key finding: "The agents that ask for help are winning" — strategic resource allocation matters more than raw output
