# Build Your First Gen AI Agent with Haystack
**Author:** Eze Lanza
**Published:** August 26, 2024

## Overview
Tutorial building a customer support chatbot using Haystack framework with Mistral-7B-Instruct model, RAG pipeline, and custom Weather API tool integration.

## Key Concepts
- Haystack open-source Python framework for LLM applications
- RAG pipeline with Apache-2 licensed Mistral model
- Custom component integration (Weather API Tool)
- Practical deployment for customer interaction
