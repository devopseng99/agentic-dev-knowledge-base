# I built InvisiBrain - a free, open-source alternative to Cluely and Parakeet AI
**Author:** Shubham Shinde
**Published:** October 30, 2025

## Overview
An invisible desktop AI assistant that operates without visual clutter during meetings and research. Open-source alternative to Cluely and Parakeet AI.

## Key Concepts
- Gemini API for rapid, context-aware responses
- Vosk AI for offline transcription (privacy-first)
- Stealth mode: disguises itself as Google Chrome in Task Manager
- Electron framework for cross-platform compatibility
- Built with Electron, JavaScript/Python
