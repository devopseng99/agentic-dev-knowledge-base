# I've built a simple k8s agent cli
**Author:** Hadi Farnoud
**Published:** April 6, 2026

## Overview
A project called KubeAgent - a CLI tool designed to manage Kubernetes cluster issues autonomously. It runs locally in the terminal with a zero-access approach and human-in-the-loop approval for risky operations. Features a knowledge base system that learns about your cluster and codebase for more reliable fixes.

## Key Concepts
- Dead simple setup, runs locally (no cluster access needed)
- Knowledge base system learns about cluster and codebase
- Safe actions done automatically, risky ones require approval
- Slack and Telegram notifications for issues
- Human-in-the-loop approach for risky operations
