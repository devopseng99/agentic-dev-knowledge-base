# This Week in Rust Trending: Storage, AI Agents, and Real-World Infra

**Author:** Linghua Jin
**Published:** December 13, 2025

## Overview
Highlights ten trending Rust repositories spanning storage systems, AI-powered development tools, blockchain infrastructure, databases, and developer productivity applications. Key themes: AI-native tooling powered by Rust, production-ready infrastructure, and enhanced developer experience.

## Key Concepts

### Featured Projects

1. **RustFS** - Distributed S3-compatible object storage with 2.3x performance vs MinIO
2. **Goose** - Extensible AI agent that modifies codebases, installs dependencies, and runs tests
3. **Polkadot SDK** - Blockchain development toolkit bundling networking, consensus, and Substrate
4. **Turso** - SQLite-compatible in-process database with async I/O and vector operations
5. **CocoIndex** - Data transformation framework for incremental processing between sources
6. **Rustlings** - Educational Rust exercises through compiler-error-fixing
7. **Magisk** - Android systemless rooting incorporating Rust for performance
8. **Yew** - WASM framework for Rust-based single-page applications
9. **Vibe Kanban** - Task orchestration board for managing AI coding agents
10. **Meetily** - Local-first AI meeting assistant with transcription via Ollama

### Key Themes
- AI-native tooling: Rust powering stateful AI infrastructure
- Production-ready infrastructure: Real workloads deployable today
- Enhanced developer experience: Lower barriers to Rust adoption
