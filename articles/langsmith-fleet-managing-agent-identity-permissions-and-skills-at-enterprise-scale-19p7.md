# LangSmith Fleet: Managing Agent Identity, Permissions, and Skills at Enterprise Scale
**Author:** Richard Dillon
**Published:** 2026

## Overview
LangChain renamed Agent Builder to Fleet in March 2026, introducing three primitives for enterprise agent management: agent identity, role-based permissions, and reusable Skills.

## Key Concepts
- Agent identity management at enterprise scale
- Role-based permissions for agent access control
- Reusable Skills as composable agent capabilities
- Fleet architecture for agent portfolio governance
- Enterprise agent lifecycle management
