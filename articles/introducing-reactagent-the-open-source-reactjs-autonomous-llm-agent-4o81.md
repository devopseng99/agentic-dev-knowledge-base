# Introducing ReactAgent: The open-source React.js Autonomous LLM Agent
**Author:** Eylon Mizrachi
**Published:** May 15, 2023

## Overview
ReactAgent is an experimental autonomous agent using GPT-4 to generate and compose React components from user stories, integrating React, TailwindCSS, TypeScript, Radix UI, and Shadcn UI.

## Key Concepts
- Generates React components from user stories
- Composes components from existing ones
- Leverages local design systems
- Applies atomic design principles
- Planned: component editing, wireframe-to-code, remote design systems

Website: reactagent.io, GitHub: github.com/eylonmiz/react-agent
