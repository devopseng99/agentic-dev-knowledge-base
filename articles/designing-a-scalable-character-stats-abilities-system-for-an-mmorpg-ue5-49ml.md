# Designing a Scalable Character Stats & Abilities System for an MMORPG (UE5)
**Author:** DomTech Gaming  **Published:** April 19, 2026

## Overview
Dev Log #8 for Magickness, an indie MMORPG in development using Unreal Engine 5. The post focuses on establishing foundational systems for character progression and combat mechanics. The developer emphasizes building interconnected, flexible systems rather than rigid, predetermined character builds.

## Key Concepts
- **Character Stats System**: Player attributes designed with scalability, capable of interacting with abilities and evolving over time
- **Abilities Framework**: Foundation for mechanics supporting discovery, combination systems, and progression-based evolution
- **System Integration**: These foundations connect to combat systems, inventory/equipment management, and magic discovery mechanics
- **Design Philosophy**: "Avoid rigid, predefined builds. Instead: allow systems to interact, allow players to experiment, allow progression to emerge"
- **Discovery-Driven RPG**: Abilities and stats are discovered through gameplay, not selected from fixed menus
- **AI in Development Workflow**: Claude used as analytical and design review tool throughout development
