# AWS Guardrails vs MCP Explained
**Author:** Ali-Funk
**Published:** January 20, 2026

## Overview
AWS Bedrock Guardrails (security/governance) and MCP (integration/capability) serve complementary roles in agentic AI.

## Key Concepts
- Guardrails: "The Bouncer stands at the door" -- filters input/output for safety
- MCP: "USB-C connects everything" -- standardized tool access
- Pipeline: User input -> Guardrails input filter -> LLM -> MCP tools -> Guardrails output filter -> response
