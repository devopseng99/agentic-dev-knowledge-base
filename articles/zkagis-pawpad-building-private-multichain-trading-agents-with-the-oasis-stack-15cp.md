# zkAGI's PawPad: Building Private, Multichain Trading Agents with the Oasis Stack
**Author:** Manav
**Published:** September 23, 2025

## Overview
PawPad combines Oasis Sapphire (EVM-compatible runtime with end-to-end encryption) and ROFL (Runtime Offchain Logic in TEEs) for private multichain trading agents with secure key management across multiple blockchain ecosystems.

## Key Concepts

### Core Architecture
- Sapphire: EVM-compatible runtime with end-to-end encryption
- ROFL: Agent logic within trusted execution environments
- Multi-curve key generation: secp256k1 (EVM/Bitcoin), Ed25519 (Solana/Aptos)

### Capabilities
- Confidential agent registries and event logging
- Private strategy storage and encrypted portfolio state
- Cross-chain signing without external bridges
- Verifiable execution through TEE attestations

### Developer Value
- Reduces trust assumptions through execution verifiability
- Maintains strategic confidentiality against MEV bots
- Genuine cross-chain operations without bridge dependencies
- Open-source contract releases planned
