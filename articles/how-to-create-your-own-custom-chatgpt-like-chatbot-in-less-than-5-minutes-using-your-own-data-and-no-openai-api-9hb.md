# How to create your own custom ChatGPT like chatbot using your own data and no OpenAI API
**Author:** Wilbert Misingo
**Published:** May 19, 2023

## Overview
Build chatbots using llama-index for document-based Q&A with free open-source models (facebook/opt-iml-max-30b) instead of paid OpenAI APIs.

## Key Concepts
- llama-index library for document indexing and retrieval
- Free Hugging Face models instead of OpenAI
- Custom LLM classes for model integration
- PromptHelper for efficient token management
- Document search rather than fine-tuning approach
