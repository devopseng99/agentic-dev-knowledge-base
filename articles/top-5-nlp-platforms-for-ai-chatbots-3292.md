# TOP 5 NLP Platforms for AI Chatbots

**Author:** Dhruv Rajkotia
**Published:** April 1, 2022

## Overview

Compares five NLP platforms for building intelligent conversational AI chatbots across messaging platforms.

## Key Concepts

### 1. Dialogflow (Google)

- Freemium pricing
- Channel-specific customizable answers
- Integrations with Messenger, Twilio, Slack
- Built-in conversation history and analytics
- Multi-language support
- Custom analytics dashboards using BigQuery and Data Studio

### 2. Wit.ai (Facebook)

- Open source
- No distinction between intents and entities
- Supports 58 languages
- Freemium model

### 3. IBM Watson

- Freemium pricing
- Automated named entity recognition
- Full query history
- Multilingual support

### 4. Rasa NLP

- Open source
- Requires programming knowledge
- On-premise or SaaS deployment
- Custom training capability

### 5. Amazon Lex

- Model version and entity tagging
- Explicit entity tagging through slots
- Supports 6 languages
