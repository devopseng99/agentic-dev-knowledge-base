# What Are Copilot Agents? Learn How to Create Them
**Author:** Lalit Mohan
**Published:** December 22, 2024

## Overview
Introduction to Copilot agents as AI-powered extensions that integrate with external data systems (legacy platforms, custom databases, third-party apps) for read-write functionality across Microsoft 365. Covers two creation approaches (Pro Code and Low Code) and four agent types in Copilot Studio.

## Key Concepts

### Creation Methods
- **Pro Code**: Developer-focused using Visual Studio or Teams Toolkit
- **Low Code**: Microsoft Copilot Studio for non-technical users

### Four Agent Types in Copilot Studio
1. **Conversation Agents**: User-Copilot dialogues
2. **Power Automate-Based Agents**: Workflow automation
3. **AI Request Agents**: Complex input interpretation
4. **OpenAI Model-Based Agents**: Custom AI tasks

### Core Benefits
- Streamline workflows by automating repetitive tasks
- Enforce uniform coding standards
- Personalize AI solutions for specific business needs
- Enhance team collaboration in real-time

### Practical Applications
Customer service, sales enablement, IT support, content creation automation.
