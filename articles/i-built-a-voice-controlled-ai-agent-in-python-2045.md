# I Built a Voice-Controlled AI Agent in Python
**Author:** Aditya Nagalkar
**Published:** April 12, 2026

## Overview
Voice-controlled AI agent with Whisper transcription via Groq, LLaMA 3.3 70B for intent recognition, modular Python architecture, and Gradio frontend with human-in-the-loop verification.

## Key Concepts
- Sub-second transcription, sub-3-second LLM responses using cloud APIs
- Reliable JSON output from LLM for structured intent
- Compound command handling
- 8-turn conversation memory
- Human-in-the-loop verification for file operations
- Performance benchmarking built-in
- Open-source, 5-minute setup
