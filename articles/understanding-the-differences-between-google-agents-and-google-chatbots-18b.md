# Understanding the Differences Between Google Agents and Google Chatbots
**Author:** Dhruv Kumar
**Published:** May 27, 2024

## Overview
A comparison of Google Chatbots (rule-based, synchronous, limited automation for FAQs) versus Google Agents (goal-oriented, asynchronous, API-connected task performers). Covers when to use each technology and introduces Vertex AI's Agent Builder platform for creating sophisticated conversational agents.

## Key Concepts

### Chatbots: Informational Transactional Output
- Rule-based output for simple FAQs and information access
- Live (synchronous) interactions restricted to single response
- Limited automation for basic, routine tasks

### Agents: Goal-Oriented Task Performers
- Interpret user intent and manage complex conversations
- Non-live (asynchronous) conversations over time
- API connectivity for booking, purchases, authentication
- Flexible handling of complex workflows

### Agent Builder Platform
Google's Vertex AI Agent Builder (previously Dialogflow CX) offers a scalable cloud-based conversational AI development platform using generative playbooks and the ReAct pattern for handling complex conversations and business workflows.
