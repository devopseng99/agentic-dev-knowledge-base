# Google's New Move! How Does the A2A Protocol Enable AI Agents to Add Friends?
**Author:** Sienna
**Published:** April 10, 2025

## Overview
Overview of Google's A2A (Agent2Agent) Protocol launched April 9, 2025 -- an open protocol enabling AI agents from different frameworks to collaborate securely. Built on HTTP, SSE, and JSON-RPC standards for easy integration.

## Key Concepts

### Core Advantages
- Cross-platform collaboration breaking technological barriers
- Increased efficiency through automated workflows
- Built on existing standards (HTTP, SSE, JSON-RPC)

### Application Scenarios
- **Recruitment**: Multiple agents handling candidate discovery, background checks, interviews
- **Customer Service**: Agents managing inquiries, backend services, follow-ups
- **Enterprise Automation**: End-to-end order processing and logistics

### Protocol Design
A2A is built on popular existing standards (HTTP, SSE, JSON-RPC), enabling easy integration with current enterprise systems without requiring new infrastructure.
