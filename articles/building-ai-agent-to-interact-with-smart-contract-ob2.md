# Building AI Agent to Interact with Smart Contract
**Author:** Ky Huynh
**Published:** August 27, 2024

## Overview
Describes MilestoneMaker, a decentralized learning platform combining blockchain and AI where personalized learning paths are generated by AI agents and progress is tracked on-chain with NFT minting for achievements.

## Key Concepts

### Workflow
Users connect via wallet (MetaMask), input learning topics, and the AI agent generates tailored learning paths. Content is stored on IPFS with the CID recorded on blockchain. Users complete tasks, check progress boxes (updating blockchain records), and take quizzes. Passing 80% of quiz questions enables NFT minting as achievement proof.

### Architecture Components
- Decentralized application (dApp)
- Backend server managing AI agents
- Blockchain smart contracts
- IPFS for content storage

### Key Features
- Personalized learning paths generated by AI agents
- Blockchain-based progress tracking
- NFT minting for achievements
- Interactive user interface
