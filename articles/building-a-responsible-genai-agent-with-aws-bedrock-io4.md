# Building a Responsible GenAI Agent with AWS Bedrock

**Author:** Shubham Kumar
**Published:** October 26, 2025

## Overview
Creating enterprise-grade AI agents using AWS Bedrock with guardrails, knowledge bases, and prompt engineering for safe, accurate, and compliant generative AI applications.

## Key Concepts

### Guardrail Components
- **PII Detection and Masking** - Automatically detects and blocks personally identifiable information
- **Content Filtering** - Restricts sensitive domains like medical advice
- **Custom Policy Enforcement** - Ensures organizational standard compliance

### Knowledge Base Options
- **Vector Store** - Customizable with automatic embedding management
- **Structured Data Store** - For databases and tables
- **Kendra GenAI Index** - For unstructured documents like PDFs

### Action Groups
Define agent capabilities: querying data, calling APIs, executing business logic, and applying guardrails.

### Key Finding
Small prompt tweaks -- like specifying format or adding context -- dramatically improved response accuracy.
