# DSPy: A Python Framework for Building Self-Learning AI Agents
**Author:** Sizan
**Published:** October 27, 2024

## Overview
DSPy enables AI agents to improve through self-learning: creating basic templates, deploying feedback loops with stronger models, iteratively refining prompts through scored corrections.

## Key Concepts
- Self-learning feedback loop: stronger model evaluates weaker model outputs
- Iterative refinement through numerical scoring
- Progressive improvement reduces prompt engineering effort
- Practical application: AI Site Reliability Engineer
