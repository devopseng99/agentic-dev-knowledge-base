# Building WiseCashAI: An Open-Source, AI-Powered Personal Finance Tracker
**Author:** Allan Ninal
**Published:** May 7, 2025

## Overview
WiseCashAI is a free, browser-based personal finance application that combines traditional envelope budgeting with artificial intelligence capabilities. The platform uses vanilla JavaScript, HTML, CSS, integrates with OpenRouter API for AI capabilities, and implements OAuth for Google authentication.

## Key Concepts
Key features include an envelope-based budgeting system, AI-powered transaction categorization using customizable smart rules, goal-setting and tracking, Google Drive integration for cross-device sync, AI-generated demo data with profession-based profiles, and receipt capture with OCR data extraction. The development roadmap includes enhanced AI forecasting, multi-currency support, and investment tracking.
