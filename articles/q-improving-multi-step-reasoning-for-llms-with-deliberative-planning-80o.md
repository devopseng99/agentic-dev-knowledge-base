# Q*: Improving Multi-step Reasoning for LLMs with Deliberative Planning

**Author:** aimodels-fyi
**Published:** June 25, 2024

## Overview
Covers the Q* framework that couples LLM language understanding with a deliberative planning module, enabling decomposition of problems into sequential steps before execution. Consistently outperforms standalone LLM baselines on logical inference, multi-hop QA, and procedural tasks.

## Key Concepts

### Core Approach
Q* integrates a neural planner that generates action plans based on problem statements and current understanding. Planning modules help systems "better understand and represent the underlying structure of the task."

### Benefits
- Improved generalization over pattern-matching approaches
- Better performance on multi-step reasoning tasks
- Structured task decomposition before execution

### Limitations
Acknowledged challenges with open-ended problems; suggests integrating more advanced planning techniques for broader applicability.
