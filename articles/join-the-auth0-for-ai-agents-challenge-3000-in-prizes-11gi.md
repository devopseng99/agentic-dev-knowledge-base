# Join the Auth0 for AI Agents Challenge: $3,000 in Prizes!
**Author:** Jess Lee

## Overview
Developer challenge to build agentic AI applications using Auth0 for AI Agents. $3,000 total prizes (3 winners x $1,000), running through October 26, 2025.

- MCP Authentication integration
- Fully functional deployed applications required
- Judged on technology implementation, use case practicality, UX, and creativity
