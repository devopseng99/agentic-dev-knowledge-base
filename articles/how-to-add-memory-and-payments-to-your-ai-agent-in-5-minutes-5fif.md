# How to Add Memory and Payments to Your AI Agent in 5 Minutes
**Author:** t49qnsx7qt-kpanks
**Published:** April 20, 2026

## Overview
MnemoPay SDK enables AI agents to store persistent memories using Ebbinghaus forgetting curve decay and process payments with automatic escrow, fraud detection, and multi-agent commerce.

## Key Concepts

### Installation

```bash
npm install @mnemopay/sdk
```

### Features
- Memory System: Memories naturally decay over time; agents recall 5 most relevant weighted by importance and reinforcement
- Agent Wallet: Automatic escrow with 1.9% fees, double-entry bookkeeping
- Multi-Agent Commerce: Direct agent-to-agent transactions with shared memory
- Fraud Detection: Velocity checks, anomaly detection, geo-risk scoring, OFAC screening
- Payment Rails: Paystack (Africa), Stripe (global), Lightning (Bitcoin)
- MCP server integration for Claude
- Apache 2.0 license
