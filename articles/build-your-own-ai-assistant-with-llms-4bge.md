# Build Your Own AI Assistant with LLMs
**Author:** Deepthi Balasubramanian & Leerish Arvind
**Published:** July 17, 2025

## Overview
Build a voice-to-notes AI assistant using Azure Speech-to-Text, Azure OpenAI, and Power Platform. Covers LLM fundamentals, prompt engineering, and responsible AI.

## Key Concepts
- LLM transformer architecture: tokenization, self-attention, output generation
- Voice-to-Notes stack: Azure Speech-to-Text + Azure OpenAI + Power Platform
- Prompt engineering for better responses
- Optional fine-tuning via Azure AI Studio
- Responsible AI: bias, cost, privacy considerations
