# What is Dremio? The Unified Lakehouse and AI Platform
**Author:** Alex Merced
**Published:** 2025

## Overview
Dremio as a unified lakehouse platform providing federated data access, intelligent caching, and semantic layer for AI agents and analytics workloads.

## Key Concepts
- Unified lakehouse combining data lake flexibility with warehouse performance
- Federated access to all data sources without ETL
- Reflections and intelligent caching for sub-second queries
- Semantic layer for AI agent data understanding
- Integration with Apache Iceberg for open table format
