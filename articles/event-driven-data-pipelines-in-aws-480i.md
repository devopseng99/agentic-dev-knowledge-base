# Event Driven Data Pipelines in AWS
**Author:** Traindex
**Published:** 2024

## Overview
Building event-driven data pipelines on AWS using managed services for real-time data processing and AI workloads.

## Key Concepts
- AWS EventBridge for event routing
- Lambda for serverless event processing
- Kinesis and MSK (managed Kafka) for streaming
- Step Functions for workflow orchestration
- S3 event triggers for data lake ingestion
- Integration with SageMaker for ML pipeline triggers
