# Top 10 Open-source AI CRM Projects with the Most GitHub Stars
**Author:** NocoBase  **Published:** December 21, 2025

## Overview
Examines how open-source CRM systems are evolving to incorporate AI. CRM is "gradually moving from a system centered on data entry and record" toward more proactive business operations.

## Key Concepts

**Native AI CRM Solutions**

1. NocoBase (20.8k stars) — "AI Employees" as embedded intelligent agents within pages and workflows; data cleaning from unstructured text, business analysis visualization, multilingual translation
2. Huly Platform (23.9k stars) — Real-time transcription via built-in assistant "Hulia"; converts voice/video to searchable text; integrates project management, communication, and CRM
3. Krayin CRM (20.6k stars) — Laravel with official AI module for LLM integration; sales content generation, lead assistance, intelligent text input

**CRMs Supporting AI Extensions**

4. Twenty (37.3k stars) — Highest; GraphQL and REST APIs; MCP-based extensions connecting to Claude; natural language CRM record management
5. ERPNext (30.6k stars) — Frappe Framework; plugins and API integrations; text generation in forms and communication
6. SuiteCRM (5.2k stars) — Mature; REST API and modular extension framework
7. Monica (23.8k stars) — Personal relationship management; external AI via REST API and Webhooks
8. Akaunting (9.4k stars) — Accounting platform; AI through REST API and extension framework
9. IDURAR (8.1k stars) — Combined ERP/CRM; APIs for external AI service connections
10. Dolibarr (6.8k stars) — Modular ERP/CRM; REST APIs and module-based extension system

**Key Distinction**
Native AI CRMs: more integrated experiences (AI part of architecture). Extensible CRMs: require more technical expertise but allow step-by-step AI adoption.
