# Dify + AgentQL: Build AI Apps with Live Web Data, No Code Needed

**Author:** Rachel-Lee Nabors (AgentQL)
**Published:** March 11, 2025

## Overview

This article announces the integration of AgentQL with Dify, a low-code platform for AI application development. Developers can now build AI applications that access and process real-time web data without requiring backend complexity.

## Key Concepts

### What You Can Build

Three live demonstration applications are highlighted:

1. A price comparison tool for real-time product pricing across websites
2. A research assistant that pulls fresh web data for answering questions
3. A news aggregator extracting and structuring content from any website

### Implementation for Agents

Users create a new Dify app, select "Agent," add tools, search for AgentQL, and select "Extract Web Data." The agent can then use either AgentQL queries or natural language prompts for data extraction.

### Implementation for Workflows

The process involves creating a Chatflow or Workflow app, adding a block, searching for AgentQL tools, and configuring input variables before running the pipeline.

## Integration Details

AgentQL now integrates directly with Dify as a tool plugin. The integration supports both agent-based and workflow-based implementations. For agents, AgentQL acts as a tool the agent can invoke when it needs real-time web data. For workflows, AgentQL nodes can be placed at any point in the processing pipeline.

The TinyFish team building AgentQL provides detailed integration guides and community support via Discord, X, and Bluesky.
