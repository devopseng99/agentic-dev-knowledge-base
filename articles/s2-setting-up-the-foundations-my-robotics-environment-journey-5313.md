# S2: Setting Up the Foundations - My Robotics Environment Journey
**Author:** Abdullateef OGUNDIPE  **Published:** April 4, 2026

## Overview
This article documents the second session of Aurora Robotics 2.0, focusing on environment setup. The author explores the critical importance of proper configuration and dependency management in robotics development, discussing multiple pathways for establishing a Linux-based workspace suitable for ROS 2 development.

## Key Concepts
- ROS 2 as the core robotics framework
- Supporting tools: Gazebo, RViz, Python, C++, Rust, Git, GitHub, VS Code, Docker, OpenCV
- Linux/Ubuntu as the preferred operating system for robotics
- Multiple setup options: WSL, dual-boot, native Ubuntu installation
- Dual-environment strategy: Ubuntu 24.04 with ROS Jazzy (WSL on Windows 11) and Ubuntu 22.04 with ROS Humble (dual-boot on Windows 10)

Key quote: "Throw your Windows out of the windows" -- emphasizing Linux's dominance in robotics development standards.
