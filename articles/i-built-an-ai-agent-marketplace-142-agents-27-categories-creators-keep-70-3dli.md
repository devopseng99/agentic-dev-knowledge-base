# I Built an AI Agent Marketplace — 142 Agents, 27 Categories, Creators Keep 70%
**Author:** Damien  **Published:** 2026-04-03

## Overview
AiPayGen, a marketplace enabling developers to monetize AI agents while maintaining favorable economics. Addresses gap in the AI agent ecosystem.

## Key Concepts

### Marketplace Stats
- 142 agents across 27 categories (finance, legal, healthcare, DevOps)
- 81,000+ API calls processed
- Agent capabilities: trading, research, coding, scraping, translation, analysis

### For Buyers
- Access via MCP (Model Context Protocol) or REST APIs
- Free testing tier with "$0.10 credits" without requiring payment information

### For Creators
Three steps: list on platform, set pricing, receive "70% of every transaction" while system handles payment processing.

### Payment Options
- Stripe for credit cards and subscriptions ($4.99–$49.99)
- Prepaid credit packs starting at $0.50
- x402 USDC cryptocurrency payments on Base, Solana, or Stellar

### Enterprise Capabilities (v1.10.0)
- Usage-based metering
- Team management with role assignments
- IP allowlisting
- Key rotation
- Audit trails
- Budget forecasting

### Unique Feature: A2A Protocol
Agents autonomously discover, negotiate with, and compensate other agents, enabling specialized subcontracting workflows.

### Technical Details
- MIT-licensed, open source
- 1,465 passing tests
- Flask and SQLite infrastructure
