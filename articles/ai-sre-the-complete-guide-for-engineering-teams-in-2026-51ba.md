# AI SRE: The Complete Guide for Engineering Teams in 2026
**Author:** Siddharth Singh
**Published:** April 24, 2026

## Overview
Comprehensive guide to AI SRE agents that autonomously triage alerts, investigate incidents, perform root cause analysis, and generate postmortems. Covers commercial and open-source tools including Aurora, K8sGPT, and HolmesGPT.

## Key Concepts

### Getting Started with Aurora (Open-Source AI SRE)
```bash
git clone https://github.com/Arvo-AI/aurora.git
cd aurora
make init
make prod-prebuilt
```

Aurora is Apache 2.0 licensed, self-hosted, supports multi-cloud (AWS, Azure, GCP, Kubernetes), and works with any LLM provider including local Ollama for air-gapped deployments.
