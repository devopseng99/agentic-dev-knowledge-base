# We Launched on Product Hunt Today: BuyWhere MCP Server for AI Agent Commerce
**Author:** BuyWhere  **Published:** 2026-05-05

## Overview
BuyWhere: open-source MCP server designed for AI agent commerce capabilities. Addresses the gap: "AI agents need structured product data. Most MCP servers skip product search."

## Key Concepts

### Core Functionality
Access to 50+ million products across six markets, enabling AI systems to perform commerce-related tasks.

### Five Primary Tools

```
search_products  — locate items across the database
get_product      — retrieve detailed product information
compare_prices   — analyze pricing across variants
get_deals        — identify current promotions
get_categories   — browse product classifications
```

### Quick Start

```bash
npx -y @buywhere/mcp-server
```

### Compatible Environments
Claude, Cursor, Cline, Continue, and VS Code.
