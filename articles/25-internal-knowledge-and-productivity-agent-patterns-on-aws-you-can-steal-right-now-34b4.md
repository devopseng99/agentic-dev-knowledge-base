# 25 Internal Knowledge and Productivity Agent Patterns on AWS You Can Steal Right Now
**Author:** Marcelo Acosta Cavalero
**Published:** April 6, 2026

## Overview
Catalog of 25 AI agent patterns for internal employee-facing use cases on AWS. Covers Amazon Q Business, AgentCore, and QuickSight across knowledge management, IT support, HR operations, engineering, and finance/operations domains.

## Key Concepts

### AWS Platforms
- **Amazon Q Business:** Permissions-aware enterprise search with native connectors and document-level ACLs
- **AgentCore:** Custom agents with orchestration, multi-step execution, fine-grained control
- **Amazon QuickSight:** Analytics, dashboarding, self-service BI

### 25 Patterns Across 5 Domains

**Knowledge Management (#026-030):** Enterprise search, policy Q&A, documentation assistance, decision logging

**IT Support (#031-035):** Help desk automation, access provisioning, incident coordination, security questionnaires

**HR Operations (#036-040):** Onboarding, benefits advisory, job matching, performance reviews, compensation analysis

**Engineering (#041-045):** Code review context, postmortem generation, dependency assessment, on-call handoffs, ADR facilitation

**Finance & Operations (#046-050):** Expense processing, procurement, budgeting, meeting summarization, status reporting

### Key Insight
Internal agent complexity resides primarily in permission models and system integration rather than sophisticated orchestration -- most successful patterns use single agents with targeted tool access.
