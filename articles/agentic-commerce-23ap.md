# Agentic Commerce
**Author:** Vedang Vatsa FRSA  **Published:** 2026-03-26

## Overview
Agentic commerce: AI agents autonomously handle shopping on behalf of users. Users describe their needs; agents execute complete transactions from discovery and price comparison through payment and delivery tracking.

## Key Concepts

### Six Protocol Standards Enabling Adoption

1. **Model Context Protocol (MCP)** — Anthropic's standard for agent-to-tool communication with persistent memory
2. **Agent-to-Agent Protocol (A2A)** — Google's system enabling direct agent-to-agent negotiation at machine speed
3. **Agent Payments Protocol (AP2)** — Google's cryptographically-signed mandate system linking intent to transactions
4. **Agentic Commerce Protocol (ACP)** — OpenAI and Stripe's in-chat purchasing integration
5. **x402 and MPP** — Crypto-native HTTP-layer payment protocols for machine-to-machine transactions
6. **Computer Use Agents** — Anthropic and OpenAI tools navigating human-built websites via mouse/keyboard control

Consumer concerns:
- Payment security: 32%

### Business Implications
Companies face restructuring across:
1. **Product Discovery** — Catalogs require semantic metadata and behavioral data embedded at infrastructure level
2. **Customer Loyalty** — Legacy points-based programs become obsolete; hyperpersonalized, intent-triggered offers replace generic promotions
3. **Commerce Platforms** — Must support structured API endpoints for inventory queries, pricing checks, and one-call checkout
4. **Payments and Fraud** — Authentication shifts from bot-prevention to enabling authorized agent transactions (Know Your Agent/KYA)
5. **In-Store Operations** — Digital and physical journeys require synchronization through agent APIs
6. **Fulfillment and Returns** — Agents automate fulfillment decisions, requiring agent-ready logistics APIs

### Requirements for Mass Adoption
- Transparent spending limits, category restrictions, and approval thresholds
- Scoped payment tokens restricted by merchant, amount, frequency
- Explainable purchasing rationales
- Government liability frameworks and consent standards (expected 2027-2028)
