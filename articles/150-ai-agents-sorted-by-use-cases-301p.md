# 150+ AI agents sorted by use cases
**Author:** Tereza Tizkova
**Published:** January 23, 2024

## Overview
A curated directory of 150+ AI agents organized by category with filtering capabilities for use cases, licensing status, and recency. The resource is available at https://e2b.dev/ai-agents.

## Key Concepts

### Featured Agent Recommendations by Category

**Coding:** Open Interpreter - terminal-controllable with web UI

**Data Analysis:** Dot and Superagent for custom agent creation

**Productivity:** Taskade (flowcharts/mind mapping) and Saga AI (writing assistance)

**Research:** Cognosys AI (daily digests) and Phind (code generation with sources)

**Build-Your-Own:** Fine (collaborative coding agents) and Crew AI (role-playing agent orchestration, similar to AutoGen or ChatDev)
