# How I Built a Self-Updating AI Sales Agent for Using Perplexity Public Rooms
**Author:** Ali Farhat
**Published:** July 24, 2025

## Overview
Building a self-updating sales agent using Perplexity Public Rooms - persistent, shared AI spaces that continuously reference indexed websites for lead qualification.

## Key Concepts
- Perplexity Public Rooms: persistent shared AI-powered spaces
- Custom prompt + source URLs = functional sales layer
- Lead qualification, pain-point identification, case study references
- No-code deployment for freelancers, startups, recruiters
- Limitations: no analytics, no API/webhook, locked to Perplexity UI
