# ChatKit Starter Template: Official OpenAI starter for building AI chat interfaces
**Author:** jQueryScript
**Published:** October 13, 2025

## Overview
Official OpenAI starter template for building AI-powered chat interfaces integrated with Agent Builder. Pre-configured Next.js application with ChatKit web component.

## Key Concepts

### GitHub Repository
https://github.com/openai/openai-chatkit-starter-app

### Key Features
- Pre-configured Next.js application with ChatKit web component
- Session creation endpoint ready for implementation
- Theme customization and starter prompts
- Response streaming and file upload handling
- Thread management built in
- Works with Agent Builder workflows
- Framework-agnostic web component

### Quick Start
Clone the repository, insert API key and workflow ID, and deploy a functional chat interface immediately.
