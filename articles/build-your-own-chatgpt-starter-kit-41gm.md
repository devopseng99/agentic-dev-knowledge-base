# Build your own ChatGPT starter kit
**Author:** Brian Douglas
**Published:** March 14, 2023

## Overview
Domain-specific AI training using embeddings, vector search with Supabase pgvectors, and Next.js prompt interface.

## Key Concepts
- Web scraping and text chunking (1000-char documents)
- OpenAI text-embedding-ada-002 for embeddings
- Supabase pgvectors for vector search
- GPT-3.5-turbo for response generation
- Workflow: scrape -> chunk -> embed -> store -> vector search -> prompt
