# OpenTelemetry for LLM Applications: A Practical Guide with LaunchDarkly and Langfuse
**Author:** Alexis Kroberson
**Published:** 2026

## Overview
Practical guide combining OpenTelemetry, LaunchDarkly feature flags, and Langfuse for LLM application observability. Langfuse surfaces prompt content, completions, and per-agent token usage.

## Key Concepts
- OpenTelemetry integration for LLM tracing
- LaunchDarkly feature flags for LLM experimentation
- Langfuse for prompt debugging and cost analysis
- Per-agent token usage monitoring
- Combined observability stack architecture
