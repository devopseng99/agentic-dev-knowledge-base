# I put a Deepseek AI agent on Fleek

**Author:** Tobiloba Adedeji
**Published:** February 3, 2025

## Overview
A quick-start guide for deploying a Deepseek AI agent on Fleek's Web3 platform in under three minutes using the "dobby" template.

## Key Concepts

### Prerequisites
1. Visit fleek.xyz/eliza
2. Create a Fleek account
3. Obtain a Telegram bot token from BotFather
4. Generate a Deepseek API key from their platform

### Deployment Steps

1. **Select Template:** Choose "dobby" from available options
2. **Choose Model Provider:** Select Deepseek from dropdown
3. **Review Details:** Confirm configuration settings
4. **Access Settings:** Click "Continue to settings"
5. **Add API Key:** Input the DEEPSEEK_API_KEY credential
6. **Deploy:** Click "Deploy agent"

The process takes approximately five steps to launch the agent live on Fleek's platform. The deployed agent can interact via Telegram using the Deepseek model for AI-powered conversations.
