# MCP Inspector: Browser-Based MCP Server Testing
**Author:** Frank Fiegel
**Published:** January 17, 2026

## Overview
Free browser-based tool for testing MCP servers -- connect to any URL and interact with tools, resources, prompts, and tasks without login or local setup.

## Key Concepts
- Privacy-first: all requests go directly from browser to MCP server, nothing proxied
- Supports tools, resources, prompts, long-running tasks
- OAuth 2.0 authentication support
- Shareable sessions stored in URLs
- Built-in demo server
- Available at glama.ai/mcp/inspector
