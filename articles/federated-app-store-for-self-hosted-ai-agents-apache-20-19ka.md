# Federated app store for self-hosted AI agents (Apache-2.0)
**Author:** Brandon Bennett
**Published:** November 5, 2025

## Overview
Decentralized Git-based marketplace for AI agents. Container isolation with egress proxy. Credential injection for API keys. Fork-based ownership model. Hash-chained audit logging. FastAPI, PostgreSQL, React, Python SDK.
