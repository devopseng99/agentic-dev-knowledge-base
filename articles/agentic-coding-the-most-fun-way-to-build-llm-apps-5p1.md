# Agentic Coding: The Most Fun Way to Build LLM Apps!

**Author:** Zachary Huang
**Date Published:** March 22, 2025

---

## Overview

This article introduces agentic coding as a development paradigm where humans design high-level system architecture while AI handles implementation details. The author created Pocket Flow, a minimalist 100-line framework designed to enable this human-AI collaboration.

## Key Concepts

### What is Agentic Coding?

The model divides labor between human creativity and AI execution:
- **Humans:** Design app architecture and workflows
- **AI:** Write implementation code based on specifications

Think of it as architect (human) and construction crew (AI) working together.

### Problems with Current Frameworks

Existing tools like LangChain suffer from:

> "high-level abstractions soon made our code more difficult to understand and frustrating to maintain"

Additional issues include dependency bloat, version conflicts, and outdated documentation.

### Pocket Flow Solution

A lightweight framework (100 lines) offering:
- Clear, understandable concepts for AI agents
- Zero vendor lock-in
- Perfect division of labor between designer and implementer

## Setup Options

| Use Case | Recommended Tool | Setup Complexity |
|----------|------------------|------------------|
| Quick questions | GPT Assistant | None required |
| Brainstorming | ChatGPT/Claude Projects | Low |
| Production apps | Cursor AI, Windsurf, Cline | Medium |

### Quick Start for Each Tool

**ChatGPT Project Setup:**
1. Create a new Project
2. Upload Pocket Flow documentation files
3. Set system prompt to guide design-first approach
4. Use Canvas for visualization

**Claude Project Setup:**
1. Start new Project
2. Upload documentation (excluding config files)
3. Configure system prompt for artifact-based design
4. Use extended thinking models for best results

**Cursor AI Setup:**
1. Clone the template repository
2. Load `.cursorrules` file automatically
3. Use Chat mode for design phase
4. Switch to Agent mode for implementation

---

## Key Takeaways

1. Agentic coding succeeds when frameworks are simple enough for AI to understand
2. Clear separation between design and implementation phases improves outcomes
3. Pocket Flow's minimalism enables better AI comprehension and collaboration
4. Different tools suit different development stages (learning -> prototyping -> production)
