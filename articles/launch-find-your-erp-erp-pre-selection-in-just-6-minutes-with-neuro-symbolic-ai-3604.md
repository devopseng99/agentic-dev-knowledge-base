# Launch: Find-Your-ERP — ERP Pre-Selection in 6 Minutes with Neuro-Symbolic AI
**Author:** Ben (Sectorlens)  **Published:** September 2, 2025

## Overview
Sectorlens launched Find-Your-ERP, a web-based tool enabling organizations to generate initial ERP system shortlists within six minutes through automated analysis combining machine learning with knowledge-based symbolic systems.

## Key Concepts

**How It Works (3 Steps)**
1. User enters their company website URL
2. Neuro-symbolic AI examines publicly accessible data to create a customized requirements profile
3. System produces a curated shortlist showing feature alignment, sector relevance, and deployment references

**Core Features & Philosophy**
- Privacy-conscious: "No data hand-off: It all starts anonymously with publicly available info"
- Explainability: The matching engine reveals rationale behind each recommendation
- Accessibility: Freemium pricing structure accommodates organizations of any size

**Technical Architecture**
Combines machine learning with "knowledge-based symbolic systems — a neuro-symbolic architecture purpose-built for software selection." Leverages curated market data accumulated over several years.

**Key Innovation: Neuro-Symbolic AI**
Traditional ERP selection involves lengthy RFP processes and consultant engagements. Neuro-symbolic AI brings:
- Structured reasoning about ERP capabilities (symbolic)
- Learning from deployment patterns and company profiles (neural)
- Explainable recommendations unlike black-box ML

**Target Audience**
- Technology leaders requiring rapid orientation
- Resource-constrained startups and SMEs
- Implementation consultants and AI practitioners

**Access:** find-your-erp.de/en
