# Meta AI Launches Llama 4: Insights, Delays, and Community Reactions
**Author:** Abhishek Dave  **Published:** April 8, 2025

## Overview
Meta has unveiled the initial Llama 4 models—Scout and Maverick—featuring native multimodal architecture and mixture-of-experts design. Scout offers 17 billion active parameters across 16 experts optimized for single H100 GPU deployment with a 10 million token context window. Maverick contains 17 billion active parameters distributed across 128 experts, emphasizing reasoning and coding capabilities. Both were distilled from Llama 4 Behemoth (288 billion active parameters), which Meta claims surpasses GPT-4.5 on STEM benchmarks. However, early user feedback on platforms like Reddit suggests skepticism about real-world performance despite favorable benchmark results.

## Key Concepts

- Mixture-of-experts (MoE) framework architecture
- Native multimodal capabilities (image understanding, long-context reasoning)
- Distillation from flagship Behemoth model
- Lightweight supervised fine-tuning and reinforcement learning
- Curriculum design for multimodal input handling
- Benchmark vs. real-world performance disconnect
