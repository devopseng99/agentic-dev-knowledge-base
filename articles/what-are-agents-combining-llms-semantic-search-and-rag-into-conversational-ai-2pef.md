# What are Agents: Combining LLMs, Semantic Search and RAG into Conversational AI

**Author:** Iulia Feroli
**Published:** October 26, 2025

## Overview

Explores how AI evolved from basic language models to sophisticated agent systems. Covers RAG foundations, why agents go beyond RAG, and how LLMs serve as a new form of user interface.

## Key Concepts

### RAG Foundation

- Large language models for text vectorization
- Semantic search for retrieving relevant documents
- Generative AI for text prediction
- RAG combining these approaches

### Why RAG Needs Agents

Two limitations: LLMs are frozen in time to training data (retraining is costly), and data privacy prevents sensitive information in training datasets.

Agents add: conversational memory across multiple exchanges and tool execution beyond simple retrieval.

### Agent Tools

Agents use "pieces of code" for searches, database lookups, calculations, and graphics generation. The LLM's job is to choose which tools to run and in what order, eliminating hardcoded workflows.

### Paradigm Shift

LLMs function as "a new form of User Interface between us and a lot of other tools and programs."
