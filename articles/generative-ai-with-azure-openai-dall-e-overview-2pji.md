# Generative AI with Azure OpenAI (DALL-E Overview)
**Author:** Daniel Gomez  **Published:** May 30, 2023

## Overview
This tutorial introduces Generative AI fundamentals and demonstrates how to leverage Azure OpenAI services for image generation using DALL-E 2. The article walks readers through creating an Azure OpenAI resource and utilizing the Azure OpenAI Studio portal to generate images from text prompts.

## Key Concepts
- Generative AI creates new content based on learned patterns from previous data
- Azure OpenAI supports three primary workloads: natural language generation, code generation, and image generation
- DALL-E 2 serves as the image generation model within Azure's ecosystem
- Azure OpenAI Studio provides a low-code interface for model interaction
- Python code can be generated directly from the "Show Code" option within Azure OpenAI Studio
