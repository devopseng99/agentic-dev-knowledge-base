# Self-Hosted Workflow Automation: Exploring Open-Source n8n Alternatives
**Author:** lifeisverygood
**Published:** 2025

## Overview
Survey of self-hosted open-source workflow automation tools as alternatives to n8n, including Temporal, Windmill, Prefect, and Apache Airflow. Evaluates each for durability, scalability, and suitability for AI agent orchestration.

## Key Concepts
Self-hosted workflow tools provide data sovereignty and customization. Key evaluation criteria: durable execution (does the workflow survive crashes?), horizontal scalability, language support, community size, and AI/agent-specific features. Temporal stands out for durable execution, while Windmill offers the best developer experience for code-heavy workflows.
