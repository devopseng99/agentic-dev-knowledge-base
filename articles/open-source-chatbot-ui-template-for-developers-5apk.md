# Open Source Chatbot UI Template for Developers
**Author:** Raghavendra Narayan Rao
**Published:** October 20, 2024

## Overview
Convo UI: open-source chatbot template with multiple conversation support, authentication, dark/light modes, markdown table rendering, custom backend integration via Flask/LangServe, and output streaming.

## Key Concepts
- Multiple conversation support via local storage and API key linkage
- Markdown table rendering solving display issues
- Custom backend integration via Flask and LangServe
- Streaming capabilities for real-time responsiveness
- Supplementary repos for RAG implementation and ML techniques
