# Rewind AI + Cursor AI = screenpipe: How We Built a High-Performance Rust Frame Streaming API
**Author:** Louis Beaumont  **Published:** November 10, 2024

## Overview
screenpipe: open-source high-performance frame streaming API built in Rust. Combines Rewind AI (screen recording/replay) and Cursor AI capabilities into an OSS tool. Provides continuous screen capture and frame streaming for AI context enrichment.

## Key Concepts
- Pure Rust implementation for performance and memory safety
- High-performance frame capture and streaming pipeline
- Integration with AI tools via frame-by-frame context injection
- Open-source alternative combining screen recording + AI assistance
