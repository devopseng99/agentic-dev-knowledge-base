# Workflows: Windmill vs n8n vs Langflow vs Temporal
**Author:** Frederic Zhou
**Published:** 2025

## Overview
Comparative analysis of four workflow tools with different strengths: Windmill (developer-first with code execution), n8n (visual automation), Langflow (AI/LLM focus), and Temporal (durable execution for distributed systems). Helps developers choose based on use case.

## Key Concepts
Each tool occupies a different niche: Windmill excels at code-centric workflows with built-in execution, n8n provides low-code visual automation, Langflow focuses on LLM pipeline orchestration, and Temporal provides enterprise-grade durable execution for mission-critical distributed systems. The right choice depends on whether you need visual building, code execution, AI orchestration, or fault-tolerant durability.
