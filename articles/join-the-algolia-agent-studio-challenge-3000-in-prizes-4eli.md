# Join the Algolia Agent Studio Challenge: $3,000 in Prizes!
**Author:** Jess Lee
**Published:** January 7, 2026

## Overview
Development challenge for building AI agents using Algolia's Agent Studio and search infrastructure. Two categories: conversational (shopping assistants, support bots) and non-conversational (proactive intelligent enhancements). Four winners ($750 each), running through February 8, 2026.

## Key Concepts
- Consumer-facing conversational AI experiences
- Consumer-facing non-conversational intelligent enhancements
- Algolia InstantSearch chat widget integration
- Contextual data retrieval for enhanced UX
- Agent Studio: https://www.algolia.com/products/ai/agent-studio
- Free Build Plan available, no credit card required
