# UGI Leaderboard on LLM Explorer
**Author:** Olga  **Published:** April 8, 2024

## Overview
The article announces a new leaderboard for evaluating uncensored general intelligence (UGI) in large language models. This addition addresses growing user interest in "uncensored models" by providing a benchmarking tool hosted on Hugging Face Spaces that assesses models ranging from 1B to 155B parameters.

## Key Concepts
- UGI Leaderboard — evaluates models on handling sensitive and controversial topics
- Uses undisclosed assessment questions to prevent training bias
- Measures two core metrics:
  - UGI score — measuring a model's knowledge of uncensored information
  - W/10 score — gauging willingness to engage with controversial topics
- Keeps specific questions confidential to maintain fairness and prevent gaming
- Hosted on Hugging Face Spaces
- Covers models ranging from 1B to 155B parameters

## LLM Explorer Complementary Tool
Offers specialized catalog with advanced filtering by:
- Model size
- Performance
- VRAM requirements
- Quantization availability
- Commercial applicability

## Assessment Approach
The leaderboard's specific test questions are kept confidential so they cannot be specifically trained against, maintaining evaluation integrity as models evolve.
