# This is what my team looks like. 7 AI agents, one dashboard, no employees.
**Author:** Rushikesh Bodakhe
**Published:** March 24, 2026

## Overview
APES OS: locally-run multi-agent AI system with 7 specialized agents, Discord integration, OpenClaw orchestration, and Supabase logging. 32 runs tracked, 66% success rate.

## Key Concepts
- 7 agents: Atlas, Koko, Caesar, Turing, Zuri, Main, APES
- Discord-based task routing to specialized agents
- Free-tier models (OpenRouter Step Flash, Gemini Flash) with fallback chains
- Supabase for task and agent log storage
- 4-step content pipeline: research -> writing -> social posts -> promotion
- Zero cloud hosting costs: entire stack runs on personal machine
- Kanban board + operations dashboard for monitoring
