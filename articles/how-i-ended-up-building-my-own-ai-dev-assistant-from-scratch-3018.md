# How I ended up building my own AI Dev Assistant from scratch
**Author:** Marco Lamina
**Published:** September 6, 2024

## Overview
Building PR Pilot, an AI-powered CLI assistant integrating GitHub, Sentry, Slack into a unified dev workflow. Reduces context-switching friction.

## Key Concepts
- Integrations: Connected services (Sentry, GitHub, Slack)
- Knowledge: Project context via Markdown hint files
- Skills: Reusable workflows defined in YAML
- Interfaces: CLI, REST API, Python SDK, Jinja Templates
