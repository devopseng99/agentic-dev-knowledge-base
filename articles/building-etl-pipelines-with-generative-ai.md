# Building ETL Pipelines With Generative AI

**Author:** Tobias Macey (host), Jay Mishra (guest)
**Published:** October 1, 2023

## Overview
A Data Engineering Podcast episode exploring how generative AI transforms ETL pipeline development, covering time efficiency, output quality, common challenges, and strategic considerations.

### AI's Role in ETL Development
- Time efficiency improvements in pipeline creation
- Output quality enhancements
- Which project types benefit most from AI integration
- Typical workflows for leveraging generative AI technologies

- Error types encountered when using AI for pipeline development
- Ongoing maintenance requirements
- Whether continuous AI involvement is necessary after initial creation

### Strategic Considerations
- Organizational concerns when experimenting with AI in data workflows
- Innovative use cases and lessons from real-world implementations
- Scenarios where AI may not be the appropriate solution

### Related Technologies
- Data Vault modeling
- Star Schema architecture
- Entity Resolution
- OpenAI/GPT models and LLaMA models
