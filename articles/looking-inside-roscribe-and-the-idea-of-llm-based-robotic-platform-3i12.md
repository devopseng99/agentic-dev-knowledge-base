# Looking inside ROScribe and the idea of LLM-based robotic platform
**Author:** robocoach  **Published:** 2023-10-23

## Overview
Explores ROScribe, a system that uses Large Language Models to generate ROS (Robot Operating System) code and configurations. Investigates the concept of LLM-based robotic platforms where natural language instructions are converted into functional robot behaviors.

## Key Concepts
- ROScribe: LLM-based code generation for ROS robotics
- Natural language to ROS node generation
- LLM as robotics programming assistant
- Prompt engineering for robot behavior specification
- Generated ROS publisher/subscriber nodes from descriptions
- Challenges of LLM-generated robot code: correctness, safety, testing
- Integration with existing ROS ecosystem
- Potential for democratizing robotics development
- Comparison of GPT-3.5 vs GPT-4 for ROS code generation quality
