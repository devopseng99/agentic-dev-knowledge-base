# Never Miss a Hackathon Again: My Calendar-Driven Automation with Runner H
**Author:** Bridget Amana
**Published:** July 6, 2025

## Overview
A calendar-driven automation using Runner H that discovers hackathons across Devpost, Dev.to, and other platforms, then creates time-blocked calendar events with details and links. Built for the Runner H AI Agent Prompting Challenge.

## Key Concepts
- Calendar automation and time-blocking
- Web scraping across multiple hackathon platforms
- Iterative prompt engineering refinement
- Integration between AI agents and calendar systems
- Passive hackathon discovery via automated weekly searches
