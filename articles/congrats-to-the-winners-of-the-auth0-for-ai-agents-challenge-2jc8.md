# Congrats to the Winners of the Auth0 for AI Agents Challenge!
**Author:** DEV Team
**Published:** November 6, 2025

## Overview
Three winners of the Auth0 for AI Agents Challenge ($3,000 total prizes) building AI agents with authentication and security.

## Key Concepts

### Winners
1. **ESG Copilot** by @omkar598 - Autonomous multi-agent AI for ESG compliance. Auth0 Universal Login, Token Vault for API keys (Climatiq, SendGrid, Pinecone), OpenFGA for fine-grained authorization
2. **Assistant0** by @async_dime - Secure enterprise AI assistant managing Gmail, Calendar, web search, documents. CIBA for high-risk operations requiring user approval
3. **Study-Flow** by @prime_299792 - Hybrid approach integrating Auth0 Token Vault into existing study management platform alongside Convex Auth

Each winner: $1,000 USD + Exclusive DEV Badge + DEV++ Membership
