# How to Build an AI Team: The Solopreneur Playbook
**Author:** neo one  **Published:** March 20, 2026

## Overview
A framework for solopreneurs to scale business operations by delegating repeatable functions to specialized AI agents rather than hiring staff. "A one person business can operate with the output profile of a ten person company" when structured this way.

## Key Concepts

1. **AI Team Definition** - Specialized agents configured for specific business functions, each with defined tools and instructions, operating autonomously with human oversight
2. **Human Judgment vs. Execution Split** - Humans retain decision-making (strategy, voice, judgment) while agents handle repeatable execution tasks
3. **Six Delegable Functions:**
   - Content and SEO
   - Customer Support
   - Outbound Outreach
   - Social Media
   - Financial Monitoring
   - Internal Research and Monitoring
4. **Implementation Process** - Five-step assignment methodology: define output, list tools, write instructions, set escalation paths, run test cycles

**Key Principle**
The functions ready for agent delegation share a common profile: they are repeatable, they have clear success criteria, and the cost of a mistake is recoverable.
