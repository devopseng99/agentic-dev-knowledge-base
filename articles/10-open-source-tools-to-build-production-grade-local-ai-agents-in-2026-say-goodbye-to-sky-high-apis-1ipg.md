# 10 Open Source Tools to Build Production-Grade Local AI Agents in 2026
**Author:** James Miller
**Published:** February 9, 2026

## Overview
Profiles 10 open-source tools with 10K+ GitHub stars: vLLM (PagedAttention inference), Ollama (simplified model deployment), LiteLLM (universal API), CrewAI (role-based agents), Continue.dev (IDE extension), Qdrant (Rust-based vector DB), AnythingLLM (RAG platform), Promptfoo (eval tool), Diffusers (image gen), Transformer.js (browser inference).

## Key Concepts
- vLLM: significantly higher throughput than standard HuggingFace library
- Qdrant: Rust-based vector database for semantic search
- Ollama: Modelfile packaging for quantized models on consumer GPUs
- CrewAI: define agent personas, goals, and backstories for multi-step reasoning
