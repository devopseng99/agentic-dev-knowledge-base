# Bitcoin is an Agent's Best Friend
**Author:** Hannah Rosenberg
**Published:** April 16, 2026

## Overview
Tutorial where an OpenClaw AI agent ("Otto") provisions and manages a Bitcoin Lightning Network node on Signet through natural language instructions, demonstrating agent interaction with payment infrastructure.

## Key Concepts
- Installing LND daemon via Go
- Creating Signet wallet with Taproot addresses
- Verifying node identity keys and network topology
- Pay-only macaroons with spending limits for security
- Future: remote signing, L402-enabled services, mainnet deployment
