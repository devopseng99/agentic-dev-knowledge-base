# 6 tools to kickstart your full-stack app with AI
**Author:** Matija Sosic
**Published:** November 28, 2023

## Overview
Explores six AI tools for full-stack web development, from code generation to design-to-code conversion.

## Key Concepts

### 1. MAGE (Magic App GEnerator)
- **URL:** https://usemage.ai/
- Free, open-source AI coding agent generating full-stack React & Node.js apps
- Generates complete codebase with React, Tailwind, Node.js, and Prisma powered by Wasp
- Nearly 30,000 apps created since launch

### 2. Aider
- **Repo:** https://github.com/paul-gauthier/aider
- Terminal-based AI pair programmer
- Integrates with existing projects, implements features with git commit automation

### 3. Shuttle AI
- **URL:** https://www.shuttle.rs/ai
- Rust backend development with GPT assistance

### 4. Supabase AI
- SQL query generation and database management assistance
- Creates tables, indexes, and database functions via AI agent

### 5. Visual Copilot
- Figma design-to-code conversion
- Generates responsive, clean, reusable frontend code from designs

### 6. GPT Pilot
- **Repo:** https://github.com/Pythagora-io/gpt-pilot
- Multi-agent system: Product Owner, Architect, DevOps, Developer
- PRD-led development, open-source, featured on GitHub Trending
- Framework: https://wasp-lang.dev/
