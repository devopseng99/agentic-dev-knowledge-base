# Post-SSLC Project: An Electromagnetic Web-Shooter Prototype
**Author:** Neeraj S  **Published:** April 2, 2026

## Overview
The developer created an electromagnetic web-shooter inspired by Spider-Man: Into the Spider-Verse. This is a "firmware-first prototype" demonstrating "the logic required to trigger a solenoid launch sequence using an ESP32."

## Key Concepts
- **Electromagnetic Induction:** Core operating principle
- **Solenoid Launch Sequence:** Primary actuation mechanism
- **MOSFET Power Stage:** Manages high-current requirements
- **Back-EMF Protection:** Flyback diode implementation to safeguard the ESP32 from inductive voltage spikes
- ESP32 as the microcontroller brain
