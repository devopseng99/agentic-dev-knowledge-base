# Advanced ChatGPT Guide - How to build your own Chat GPT Site
**Author:** notHanii
**Published:** December 23, 2022

## Overview
A guide to building a ChatGPT-like site covering prompt engineering, OpenAI's Playground, and creating a Node.js API with Express that queries OpenAI's API.

## Key Concepts
- Prompt Engineering: Structuring queries strategically to obtain desired AI outputs
- Model Selection: Understanding differences between OpenAI's various models (text-davinci-003)
- Few-Shot Learning: Providing examples to guide AI behavior
- Express.js Backend: Building server infrastructure for API requests
- React Frontend: Creating user interfaces for AI interaction
- Role-Playing Prompts: Engineering AI to assume personas
- API Integration: Connecting applications to OpenAI's completion endpoints
