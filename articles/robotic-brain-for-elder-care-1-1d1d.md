# Robotic Brain for Elder Care 1
**Author:** susanayi  **Published:** March 30, 2026

## Overview
This article introduces a robotic system designed to support elderly individuals requiring assistance. The author is building an MVP using a decoupled architecture where Unity handles physical simulation and Python manages high-level decision-making. Development currently occurs in a simulated home environment with three experimental zones.

## Key Concepts
- **Target Use Case:** Support for paralyzed, bedridden, or 24/7 care-dependent individuals
- **MVP Strategy:** Starting with healthy users in ideal scenarios for rapid iteration
- **Architecture Approach:** Clean separation between virtual simulation (Unity) and logical processing (Python)
- **Simulation Environment:** Three zones -- Living Room, Workspace, Kitchen -- equipped with virtual cameras for perception testing
- **Data Flow:** User behavior in Unity transmits to Python for AI decision-making

Part of a series building a training-free home care robot with VLM-based activity recognition.
