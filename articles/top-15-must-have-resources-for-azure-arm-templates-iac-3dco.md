# Top 15 Must-Have Resources for Azure ARM Templates (IaC)
**Author:** vAIber  **Published:** June 21, 2025

## Overview
A curated collection of 15 essential resources for mastering Azure Resource Manager templates and Infrastructure as Code on Azure, spanning from foundational concepts to advanced deployment strategies.

## Key Concepts
- **Azure Resource Manager (ARM) Templates**: Native Azure tool for defining infrastructure in declarative JSON format
- **Infrastructure as Code (IaC)**: Practice of managing cloud infrastructure through code rather than manual processes
- **Declarative Approach**: Defining desired end state rather than procedural steps
- **Idempotent Templates**: ARM templates that produce consistent results regardless of execution count
- **Bicep Language**: Newer, more readable alternative to JSON for ARM templates
