# GhostPilot: GPS-Denied Drone Navigation with Visual SLAM and Agentic AI
**Author:** Aman Sachan  **Published:** 2026-04-30

## Overview
Earlier iteration of the GhostPilot project documenting GPS-denied drone navigation using Visual SLAM combined with agentic AI for mission planning. Covers the core architecture for indoor and GPS-denied autonomous flight.

## Key Concepts
- GPS-denied navigation using camera-based localization
- Visual SLAM (Simultaneous Localization and Mapping)
- Agentic AI for high-level mission planning and task decomposition
- ROS2 integration for robotics middleware
- Indoor drone flight autonomy
- Real-time pose estimation from visual input
- Mission execution pipeline: natural language to flight commands
- Safety constraints and failsafe mechanisms
