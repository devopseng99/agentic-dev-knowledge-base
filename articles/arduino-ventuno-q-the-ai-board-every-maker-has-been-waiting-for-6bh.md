# Arduino VENTUNO Q: The AI Board Every Maker Has Been Waiting For
**Author:** Bit to Build  **Published:** 2026-05-04

## Overview
Announces Arduino's newest development board created through partnership with Qualcomm, combining neural processing capabilities with real-time motor control. Targets makers interested in AI, robotics, and IoT by bridging the gap between traditional Arduino projects and advanced AI applications.

## Key Concepts
- Neural Processing Unit (NPU): Up to 40 TOPS for handling AI workloads
- STM32H5 Microcontroller: Handles real-time motor control with minimal latency
- Raspberry Pi Hat Connector: Ensures ecosystem compatibility
- Qualcomm Dragonwing IQ8 Series: The processor powering the board
- Launch Timeline: Expected Q2 2026 (April-June)
- Target Audience: Robotics enthusiasts, IoT developers, and edge AI experimenters
- Qualcomm's Arduino acquisition creates accessibility to powerful AI processing without expensive enterprise hardware
- Dual processor architecture: separate processors handle AI/networking and real-time hardware control simultaneously
- Hybrid Operating Modes: Standalone SBC mode or connected to a laptop
- Edge AI Processing: Local face detection without cloud dependency
- Visual block programming via Arduino App Lab
