# Stable Diffusion 3.5 Large How To Use Tutorial With Best Configuration and Comparison With FLUX DEV
**Author:** Furkan Gözükara  **Published:** October 26, 2024

## Overview
This tutorial compares two advanced image generation models—Stable Diffusion 3.5 Large and FLUX DEV—providing practical guidance on optimal configurations. The creator tested 62 prompts on all experiments to find best Sampler + Scheduler and evaluated multiple precision settings (FP16 vs Scaled FP8).

## Key Concepts

- **Model Architecture:** SD 3.5 Large is a full, non-distilled model with open licensing, unlike FLUX
- **Precision Variants:** FP16, Scaled FP8, and T5 XXL configurations compared
- **Fine-tuning Potential:** SD 3.5 supports DreamBooth and LoRA training workflows
- **Interface:** SwarmUI deployment on Windows, Massed Compute, and RunPod platforms
