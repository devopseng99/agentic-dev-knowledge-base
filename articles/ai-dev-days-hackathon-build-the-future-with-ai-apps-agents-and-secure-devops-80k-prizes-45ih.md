# AI Dev Days Hackathon: Build the Future with AI Apps, Agents, and Secure DevOps ($80k+ Prizes)
**Author:** Ayub Henry
**Published:** February 19, 2026

## Overview
A five-week global hackathon inviting developers to build AI agents, agentic DevOps workflows, and intelligent applications using Microsoft's AI platform, GitHub, VS Code, and Azure services. Total prize pool exceeds $80,000.

## Key Concepts
- AI agents via Microsoft Agent Framework
- Agentic DevOps workflows using GitHub Copilot Agent Mode
- Multi-agent systems powered by Model Router
- Enterprise AI applications with Microsoft Foundry
- Agent-to-agent (A2A) or MCP-connected systems using Azure MCP
- Next-generation SRE workflows with Azure SRE Agent

### Prize Structure
- Grand Prize (overall): $20,000 + Build 2026 ticket + mentorship
- Grand Prize (agentic DevOps): $20,000 + Build 2026 ticket + mentorship
- Microsoft Foundry Project: $10,000 + Build 2026 ticket
- Enterprise Solution: $10,000 + Build 2026 ticket

Solutions must incorporate Microsoft Foundry, Microsoft Agent Framework, Azure MCP, or GitHub Copilot Agent Mode, deploy to Azure services, and maintain public GitHub repositories.
