# From Vertex AI to Agent Platform: Why Google's Rebrand Is Actually a Philosophical Shift

**Author:** MakendranG
**Published:** April 26, 2026

## Overview

Argues that Google's renaming of Vertex AI to Gemini Enterprise Agent Platform represents a fundamental philosophical shift: agents are not AI features but software workloads requiring proper enterprise infrastructure.

## Key Concepts

### Four Core Pillars

1. **Build** - Enhanced ADK with graph-based agent networks
2. **Scale** - Agent Runtime with sub-second cold starts
3. **Govern** - Agent Identity and Agent Gateway for security/permissions
4. **Optimize** - Agent Observability, Simulation, and Evaluation tools

### Notable Features

- Over 200 models in Model Garden including competitors (Claude, Llama, Mistral, Grok)
- 6+ trillion monthly tokens through ADK
- Model Armor for prompt-injection protection
