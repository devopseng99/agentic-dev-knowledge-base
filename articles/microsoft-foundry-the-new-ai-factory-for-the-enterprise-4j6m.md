# Microsoft Foundry: The New AI Factory for the Enterprise
**Author:** Seenivasa Ramadurai
**Published:** November 30, 2025

## Overview
Analysis of Microsoft Foundry (evolved from Azure AI Studio) as a comprehensive enterprise AI platform built on four pillars: Foundry Models, Foundry Agent Service, Foundry IQ, and Control Plane. Covers the Microsoft Agent Framework unifying Semantic Kernel and AutoGen.

## Key Concepts

### Four Core Pillars

1. **Foundry Models**: Centralized catalog with models from OpenAI, Anthropic, Cohere, NVIDIA. Automatic model routing optimizing for speed, accuracy, and cost.

2. **Foundry Agent Service**: Build individual agents, teams, or multi-step workflows using Microsoft Agent Framework (released October 2025). Unifies Semantic Kernel and AutoGen for Python and .NET.

3. **Foundry IQ**: Reasoning layer built on Azure AI Search that synthesizes across organizational data while respecting security boundaries. Beyond traditional RAG.

4. **Control Plane**: Enterprise governance with evaluations, Azure Monitor observability, compliance tools (GDPR, HIPAA), GitHub Advanced Security integration.

Over 60,000 organizations already using Azure AI services.
