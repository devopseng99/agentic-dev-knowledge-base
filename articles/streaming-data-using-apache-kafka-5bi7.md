# Streaming Data Using Apache Kafka
**Author:** Mwanaisha Suleiman

## Overview
Introduction to data streaming with Apache Kafka covering basic producer/consumer patterns and topic management.

## Key Concepts
- Kafka as distributed streaming platform for high-throughput data
- Producer-consumer model for decoupled data flow
- Topics and partitions for organized data streaming
- Consumer groups for parallel processing
- Offset management for message tracking
