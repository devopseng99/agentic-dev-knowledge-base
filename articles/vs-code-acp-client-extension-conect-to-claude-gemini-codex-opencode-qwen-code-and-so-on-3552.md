# VS Code ACP Client Extension
**Author:** Jun Han
**Published:** February 12, 2026

## Overview
VS Code extension enabling integration with multiple AI coding agents via the Agent Client Protocol (ACP), similar to how LSP standardizes language servers.

## Key Concepts
- 8 pre-configured agents: GitHub Copilot, Claude Code, Gemini CLI, Qwen Code, OpenCode, Codex CLI, Qoder CLI, Auggie CLI
- Interactive chat panel with Markdown rendering and tool execution display
- File system integration for workspace read/write
- Terminal execution with output display
- Permission management with configurable auto-approval policies
- Protocol logging for inspecting ACP JSON-RPC messages
