# Build your own Posthog - PART 2
**Author:** Deepraj Baidya
**Published:** September 4, 2025

## Overview
Building an analytics system's API Gateway infrastructure using Docker, PostgreSQL, RabbitMQ, Express.js, and TypeScript.

## Key Concepts
- API Gateway as centralized entry point for request routing
- Docker Compose with PostgreSQL and RabbitMQ
- Express.js/TypeScript proxy middleware for service routing
- CORS management and request/response logging
- Monorepo project structure
