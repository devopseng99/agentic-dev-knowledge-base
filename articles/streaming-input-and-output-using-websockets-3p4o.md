# Streaming Input and Output Using WebSockets
**Author:** AG2 AI
**Published:** 2025

## Overview
Using WebSockets for bidirectional streaming of AI agent input and output, enabling real-time interactive experiences.

## Key Concepts
- WebSocket-based bidirectional communication for agents
- Streaming both input (user messages) and output (agent responses)
- Real-time event handling for interactive AI applications
- Low-latency communication compared to HTTP polling
- Integration with agent frameworks for production streaming
