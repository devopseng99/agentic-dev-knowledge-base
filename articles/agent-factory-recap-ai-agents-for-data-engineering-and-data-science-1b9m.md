# Agent Factory Recap: AI Agents for Data Engineering and Data Science

**Author:** Lucia Subatin (Google AI)
**Published:** January 22, 2026

## Overview

Recap of "The Agent Factory" podcast covering production-ready AI agents for data professionals. Highlights Gemini Computer Use, CodeMender security agent, BigQuery data engineering agents, and Spanner graph database applications.

## Key Concepts

### Gemini Computer Use Model

Enables agents to see and interact with computer screens by taking screenshots, deciding on UI actions, and executing them with safety checks requiring human confirmation.

### CodeMender

AI security agent that patches vulnerabilities and rewrites code. Uses Gemini Deep Think and has upstreamed 72 security fixes to open-source projects.

### BigQuery Data Engineering Agent

Automates data pipeline tasks: adding computed fields, creating time dimension tables, and generating data quality assertions automatically.

### Data Science Agent (Colab Enterprise)

Performs anomaly detection using isolation forest models, automated data preprocessing, and visualization with root cause insights.

### Spanner Graph Database + Comics

Combines Spanner's graph database built from documentation with a knowledge agent that queries the database and another agent that generates image prompts, creating six-panel comic strips explaining database concepts.
