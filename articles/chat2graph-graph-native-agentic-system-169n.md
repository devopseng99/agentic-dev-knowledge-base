# Chat2Graph: Graph Native Agentic System

**Author:** Florian (TuGraph Open Source Lead, Ant Group)
**Published:** May 4, 2025

## Overview

Introduces Chat2Graph, a graph-native agentic system powered by Ant Group's TuGraph team that establishes a multi-agent framework directly on graph database infrastructure. Leverages graph data structure advantages for reasoning, planning, memory management, and tool application.

## Key Concepts

### Core Philosophy

Deeply implements the "Graph + AI" concept through integrated design of graph computing and artificial intelligence technologies within agent systems.

### Bidirectional Enhancement

The system merges graph database capabilities with AI technology, using relationship modeling and interpretability to strengthen AI agent functions.
