# Pokemon Go Data Built the Robot Navigation Maps We Use
**Author:** Simon Paxton  **Published:** March 26, 2026

## Overview
The article examines how Niantic leveraged player-contributed scans from Pokemon Go to construct a pedestrian-perspective geospatial model now powering autonomous delivery robots. Rather than accidental AI training, this represents deliberate infrastructure development by a private company using public space data.

## Key Concepts
- **Large Geospatial Model (LGM):** Billions of geolocated images enabling centimeter-scale localization
- **Visual Positioning System (VPS):** Machine recognition technology accurate to the centimeter, replacing GPS in urban environments
- **Pedestrian-perspective mapping:** Data collected from human eye-level rather than vehicle-mounted cameras
- **Infrastructure-grade accountability:** The argument that geospatial AI maps should receive civic governance equivalent to traditional public infrastructure
- **Coverage inequities:** Risk that underserved neighborhoods remain algorithmic blind spots in mapping databases

Key quote: "The map wasn't a byproduct. It's the point."
