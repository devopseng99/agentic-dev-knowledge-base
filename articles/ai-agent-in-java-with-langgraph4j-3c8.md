# AI Agent in Java with LangGraph4j
**Author:** bsorrentino
**Published:** May 21, 2024

## Overview
LangGraph4j brings the flexibility of LangGraph to develop stateful, multi-agent applications with LLMs in Java. Leverages LangChain4j for LLM integration. Demonstrates agent executor pattern and multi-modal image-to-diagram conversion.

## Key Concepts
- Agent executor: runtime calling agents, executing actions, returning outputs in a cycle
- Image to Diagram: multi-modality with agents analyzing images, translating to PlantUML
- Error-correction loops for PlantUML validation
- Uses LangChain4j framework for Java LLM provider integration
