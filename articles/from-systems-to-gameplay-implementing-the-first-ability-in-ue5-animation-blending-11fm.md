# From Systems to Gameplay: Implementing the First Ability in UE5 (Animation Blending)
**Author:** DomTech Gaming  **Published:** April 22, 2026

## Overview
Dev Log #9 marking a pivotal transition in the Magickness MMORPG development — from building systems to creating actual gameplay. Implements the first unlockable combat ability with a focus on seamless animation blending with character locomotion.

## Key Concepts
- **Ability Activation Logic**: Core mechanics for triggering and executing abilities tied to player input
- **Animation Blending with Locomotion**: Smooth transitions between movement and ability animations without snapping or input lag — critical for game feel
- **Combat System Integration**: Connecting abilities to existing combat framework and character stats system
- **Ability Design Philosophy**: Abilities are discovered through gameplay and combined into new mechanics rather than being fixed loadouts
- **Early Gameplay Testing**: Transition from theoretical systems to testable, visible gameplay — the first major milestone

## Development Milestone
"This is the shift from 'building systems' to 'creating gameplay'" — represents the project leaving the foundation phase and entering the game-feel iteration phase.
