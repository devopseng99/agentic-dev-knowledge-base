# Building a Simple Stock Insights Agent with Gemini 2.0 Flash on Vertex AI
**Author:** Thomas Chong
**Published:** March 2, 2025

## Overview
Stock analysis agent using Gemini 2.0 Flash on Vertex AI with Google Search for real-time financial data and Gradio UI for interactive analysis.

## Key Concepts
- google-genai library for structured JSON responses
- Company overviews, news, and trend predictions
- Gradio interface for stock ticker/company name input
- Native tool-use capabilities for finance applications
- Low latency model suitable for real-time analysis
