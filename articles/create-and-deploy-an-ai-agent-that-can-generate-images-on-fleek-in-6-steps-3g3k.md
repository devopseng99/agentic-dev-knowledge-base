# Create and deploy an AI agent that can generate images on Fleek in 6 steps

**Author:** Tobiloba Adedeji
**Published:** January 28, 2025

## Overview
No-code deployment of an AI image generation agent on Fleek using Telegram. Deploys in under 20 seconds using the C-3PO template with OpenAI API.

## Key Concepts

### Prerequisites
- Fleek account (fleek.xyz/eliza)
- OpenAI API key
- Telegram bot from BotFather

### Steps
1. Navigate to fleek.xyz/eliza
2. Select "Start with a template"
3. Choose C-3PO template
4. Confirm Telegram client + image generation plugin enabled
5. Configure Telegram bot token and OpenAI API key
6. Click "Deploy agent" -- live in under 20 seconds
