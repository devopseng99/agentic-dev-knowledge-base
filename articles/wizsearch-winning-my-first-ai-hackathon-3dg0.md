# WizSearch: Winning My First AI Hackathon
**Author:** Sanjay
**Published:** June 11, 2024

## Overview
WizSearch won "The Future of AI is Open" hackathon. It's an AI assistant that retrieves and synthesizes information from the internet with source citations, using open-source tools including Snowflake Arctic and Streamlit.

## Key Concepts
- LLM: Snowflake Arctic
- Embeddings: Snowflake Arctic Embedding Model
- Search: Tavily
- Vector Database: Qdrant
- Observability: Langfuse
- UI: Streamlit
- Addressed: unpredictable LLM outputs, retrieval accuracy, prompt injection security
- Future: agentic capabilities, voice interaction, RAGAS evaluation
