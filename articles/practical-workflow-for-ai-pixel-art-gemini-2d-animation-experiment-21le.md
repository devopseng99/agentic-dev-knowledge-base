# Practical Workflow for AI Pixel Art: Gemini 2D Animation Experiment
**Author:** Kenji | Unity Game Dev  **Published:** March 14, 2026

## Overview
This article documents using Google's Gemini "Nano Banana 2" image generation model to create sprite sheets for 2D action games in Unity. The author explores the practical viability of AI-generated pixel art for game development, highlighting both capabilities and limitations.

## Key Concepts

- **Infinite regeneration capability** ("gacha" system) for iterative refinement
- Consistency in character design across multiple generated action sequences
- Partial modification instructions while preserving base designs
- Frame-by-frame prompt breakdown for complex animations
- Manual background removal workflow (solid color → transparency in Photoshop)

**Effective for:** Simple movements (idle, walking), character consistency, rapid prototyping

**Challenging for:** Complex sequential actions (combat combos), maintaining animation fluidity, transparent background output
