# Hugging Face Launches Free AI Agents Course with Certification!

**Author:** Riana Azad
**Published:** February 16, 2025

## Overview

Hugging Face introduced a free AI Agents course with certification covering popular frameworks including SmolAgents, LangGraph, and LlamaIndex.

## Key Concepts

### Course Structure

- **Chapter 0 (Onboarding)** - Tool and platform setup
- **Chapter 1 (Agent Fundamentals)** - Core concepts: tools, actions, observations, LLMs with practical Python applications
- **Chapter 2 (Frameworks)** - SmolAgents, LangGraph, and LlamaIndex
- **Chapter 3 (Use Cases)** - Real-world applications and community contributions
- **Chapter 4 (Final Assignment)** - Agent development, benchmarking, and leaderboard competition

### Requirements

- Basic Python knowledge
- Foundational LLM understanding
- Free Hugging Face account

### Certification Levels

1. **Fundamentals Certificate** - Upon completing Unit 1
2. **Full Certificate of Completion** - After finishing Unit 1, completing use-case assignments, and passing the final challenge

## Framework Coverage

The course explores how to build agents using:
- **SmolAgents** - Hugging Face's lightweight agent framework
- **LangGraph** - State machine-based agent orchestration
- **LlamaIndex** - Data-aware agent development

All course materials are available at no cost.
