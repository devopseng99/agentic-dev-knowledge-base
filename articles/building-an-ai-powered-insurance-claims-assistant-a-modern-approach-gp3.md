# Building an AI-Powered Insurance Claims Assistant: A Modern Approach
**Author:** Ramandeep Singh
**Published:** May 29, 2025

## Overview
Claims Buddy is an intelligent chatbot system using three specialized agents (FAQ Agent, Claims Agent, Handoff Agent) to streamline insurance claim processes. Built with FastAPI, React, WebSockets, and OpenAI's Agents SDK.

## Key Concepts
The multi-agent architecture handles different claim scenarios with real-time WebSocket communication. Backend deployed on Render.com, frontend on Vercel. Uses OpenAI's latest models for response generation.
