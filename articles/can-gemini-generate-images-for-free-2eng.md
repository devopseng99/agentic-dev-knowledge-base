# Can Gemini Generate images for free?
**Author:** Dev Author  **Published:** 2025

## Overview
Article addressing whether Google's Gemini can generate images for free, covering Imagen 4-powered image generation capabilities available to all Gemini app users.

## Key Concepts
- All Gemini app users have "general access" to Imagen 4-powered image generation at no cost
- Includes the ability to create new images and perform basic editing tasks
- Background removal and sticker creation included in free tier
- Gemini 2.0 Flash experimental model available for developers via API with free tier

## Key Findings
Google has made image generation accessible through the Gemini app for free users, though API access for developers may have usage quotas and rate limits on the free tier.
