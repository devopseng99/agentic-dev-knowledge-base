# I built a Telegram bot that generates AI images using Stable Diffusion — here's how
**Author:** Rahul K  **Published:** 2026-04-27

## Overview
The developer created a Telegram bot enabling direct AI image generation within chat. The project gained 128 clones in two weeks without promotion. Users send text prompts; the bot returns images generated by SDXL/Juggernaut XL models via ComfyUI.

## Key Concepts

### Technology Stack
- **python-telegram-bot** — Telegram API management
- **ComfyUI** — Image generation backend
- **SDXL/Juggernaut XL** — AI models
- **Websockets** — Bot-to-ComfyUI connectivity
- **Oracle Cloud ARM** — 24/7 free hosting

### Technical Architecture

```python
async def generate_image(prompt: str) -> bytes:
    workflow = build_workflow(prompt)
    async with websockets.connect(COMFYUI_WS) as ws:
        await queue_prompt(workflow, ws)
        image_data = await wait_for_output(ws)
    return image_data

async def handle_message(update, context):
    prompt = update.message.text
    await update.message.reply_text("Generating... ")
    image = await generate_image(prompt)
    await update.message.reply_photo(image)
```

### Queue Management
Simultaneous user requests require queuing. The solution uses asyncio queues for sequential processing, notifying users of their queue position.
