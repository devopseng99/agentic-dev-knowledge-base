# Building Janus: An AI-Powered Helpdesk That Makes Customer Support Smarter
**Author:** K Om Senapati
**Published:** October 28, 2025

## Overview
Janus is an AI-driven helpdesk platform with automatic ticket classification by type, priority, and category. Built with Streamlit, MindsDB for AI agents, ChromaDB for vector storage, and Nebius as the LLM provider.

## Key Concepts
Users submit tickets and receive immediate AI responses. The admin dashboard provides analytics and knowledge base management, reducing response times and enabling human agents to focus on complex cases.
