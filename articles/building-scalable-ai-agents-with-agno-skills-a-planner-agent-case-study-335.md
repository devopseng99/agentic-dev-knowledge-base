# Building Scalable AI Agents with Agno Skills: A Planner Agent Case Study
**Author:** Harish Kotra
**Published:** January 23, 2026

## Overview
Introduction to Agno Skills - reusable modular components that package specialized capabilities for AI agents. Case study: Planner Agent that decomposes goals into structured tasks with dependencies and roles.

## Key Concepts
- Skills as directories with SKILL.md instruction files
- Separation of domain logic (skills) from agent logic
- Model agnostic: same skill works across GPT-4o, Ollama Gemma3
- Task decomposition with dependency mapping and role assignment
