# I Built a Tableau Dashboard with an AI Agent
**Author:** suddhasheel bhatt
**Published:** April 5, 2026

## Overview
Building Tableau dashboards using Twilize MCP server for programmatic workbook creation through function calls, making workflows reproducible and versionable.

## Key Concepts
- Twilize: MCP server as programmatic layer for Tableau
- 8 steps: capabilities check, workbook creation, 3 worksheets, dashboard assembly, validation, save
- Under 3 minutes for complete .twb file
- Built-in best practices and explicit unsupported-feature acknowledgment
- Natural language interfaces as next phase of data visualization
