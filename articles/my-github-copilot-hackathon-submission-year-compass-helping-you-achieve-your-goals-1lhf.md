# My GitHub Copilot Hackathon Submission: Year Compass
**Author:** Erick Christian
**Published:** January 18, 2025

## Overview
Year Compass is a web app that uses AI to break down New Year's resolutions into actionable steps. Built for the GitHub Copilot Challenge: New Beginnings, using GitHub Copilot and GitHub Models.

## Key Concepts
- AI-generated breakdown of goals into manageable tasks
- GitHub Copilot Edits for refactoring and feature additions
- GitHub Models integration for step generation
- Checkbox progress tracking with visual progress bar
- Browser-based local storage (no backend)
- Milestone celebration notifications at 25% and 50% completion
