# LLAMA 4: Meta's New AI Models Break Context Length Records and Challenge Industry Leaders
**Author:** Rustamjon Akhmedov  **Published:** April 6, 2025

## Overview
Meta has released LLAMA 4, an open-source LLM suite featuring unprecedented capabilities. The article highlights three models: Scout (17B active parameters, 10M token context), Maverick (400B total parameters, 128 experts), and Behemoth (2T parameters, still in training). The most significant advancement is Scout's 10-million token context window.

## Key Concepts

1. **Mixture-of-Experts Architecture:** Each token activates only necessary parameters rather than all parameters, enabling efficiency (e.g., Maverick uses 17B of 400B parameters per token)
2. **Extended Context Window:** Scout's 10M token capacity far exceeds competitors; enables processing entire codebases and multi-document analysis simultaneously
3. **Multimodal Processing:** All models handle text, images, and video natively
4. **Performance Benchmarks:** Maverick achieves 69.8 on GPQA Diamond versus GPT-4's 53.6; ranks alongside Gemini 2.5 Pro with ~420 ELO score
5. **Open-Source with Restrictions:** Available locally but requires permission for 700M+ user applications; attribution required
