# pi-mono: Minimalist & High-Performance AI Coding Agent
**Author:** WonderLab
**Published:** May 2, 2026

## Overview
TypeScript monorepo by libGDX founder. Custom differential rendering TUI eliminating terminal flicker. System prompt under 1000 tokens. Four atomic tools: read, write, edit, bash. YOLO mode for autonomous execution. Compares favorably to Claude Code in speed and transparency.

## Key Concepts
- Virtual DOM-inspired differential rendering for terminal
- Bash-only philosophy: master shell commands to accomplish anything
- Multi-model switching across LLM providers
- 430+ GitHub stars, MIT licensed, npm: @mariozechner/pi-coding-agent
