# GitHub Copilot SDK vs Azure AI Foundry Agents: Which One Should Your Company Use?
**Author:** Ve Sharma (Solution Engineer at Microsoft)
**Published:** February 10, 2026

## Overview
Comparison guide between GitHub Copilot SDK (developer-centric agentic foundation) and Azure AI Foundry Agents (business-wide automation platform). Covers decision framework, use cases, compliance, and when to use both.

## Key Concepts

### GitHub Copilot SDK
- Production-tested agentic foundation
- Handles context management, tool orchestration, MCP integration, model routing
- Best for: CLI tools, IDE extensions, developer workflows

### Azure AI Foundry Agents
- Multi-agent orchestration platform
- Built-in data connectors (SharePoint, SQL, M365)
- Customizable model selection
- Best for: Customer support, document processing, business automation

### Decision Framework
- **Use Copilot SDK**: Developer tools, internal workflows, ops automation
- **Use Foundry**: End-user products, non-developer apps, multi-agent systems
- **Use Both**: Enterprises needing both developer and business-facing solutions

### Compliance
- GitHub: SOC 2 Type II, ISO 27001, IP indemnification
- Azure: Additional VNet isolation, customer-managed encryption
