# What is OpenClaw? A Self Hosted AI Assistants
**Author:** Sarvar Nadaf
**Published:** March 30, 2026

## Overview
Node.js gateway routing between messaging platforms and LLMs via WebSocket. SQLite local storage. Multi-channel: WhatsApp, Telegram, Discord, Slack, Teams, Signal, Matrix, IRC. Skills plugin framework with ClawHub registry. Per-session Docker sandboxing.
