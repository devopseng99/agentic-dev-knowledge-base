# GitHub + DEV 2023 Hackathon Winners, Announced!
**Author:** Brian Bethencourt
**Published:** June 13, 2023

## Overview
Winners of the GitHub + DEV 2023 Hackathon, which challenged developers to build applications using GitHub Actions and GitHub Codespaces across five categories.

## Key Concepts

### Grand Prize Winners
1. **Maintainer Must-Haves:** DeepL Translate Github Action by Estee Tey
2. **DIY Deployments:** OpenCommit - GitHub Action for meaningful commit messages by Dima Sukharev
3. **Interesting IoT:** MooptOS - Open IoT Platform for Decentralized Urban Farming by Kai Chew
4. **Phone Friendly:** Deploy pygames to GitHub Pages with WebAssembly by Santhosh
5. **Wacky Wildcards:** Tetris generated from GitHub by Nabil Alamin

### Notable Runner-Ups
- AdaGPT: AI support for Issues and Pull Requests
- AI-powered changelog updates on Slack with GitHub Actions
- Effortless Documentation of Python Code with GitHub Actions and GPT3
- FruitifyMe: Deep learning fruitifier using GitHub Actions, AWS, TensorFlow, React.JS
