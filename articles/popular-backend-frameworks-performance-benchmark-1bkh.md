# Best popular backend frameworks by performance benchmark (2025)
**Author:** Tuan Anh PHAM
**Published:** June 2, 2024

## Overview
TechEmpower Round 23 (Feb 2025) Fortunes benchmark with PostgreSQL. Rankings: C# Asp.net (36.3x baseline), Go Fiber (20.1x), Rust Actix (19.1x), Java Spring (14.5x), Node Express (4.7x), Ruby Rails (2.5x), Python Django (1.9x), PHP Laravel (1.0x baseline). Java Spring improved 9x year-over-year.
