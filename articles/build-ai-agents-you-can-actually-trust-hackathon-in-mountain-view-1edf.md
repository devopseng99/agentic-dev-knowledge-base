# Build AI Agents You Can Actually Trust — Hackathon in Mountain View
**Author:** Harry Kimpel
**Published:** March 11, 2026

## Overview
A free, in-person hackathon at Microsoft Mountain View Campus where participants build WanderAI, an AI-powered travel planning assistant, and make it production-ready with observability and security controls. Part of Microsoft's What The Hack series.

## Key Concepts
Eight progressive challenges covering agent architecture, Flask web apps with tool calling, OpenTelemetry instrumentation, New Relic integration, dashboards and alerting, LLM quality gates, Microsoft Foundry Guardrails, and prompt injection detection.

### Technical Stack
- Microsoft Agent Framework (multi-agent orchestration)
- OpenTelemetry (traces, metrics, logs)
- New Relic (telemetry visualization and alerting)
- Azure (cloud infrastructure)
- Python + Flask (application layer)
- GitHub Codespaces (development environment)
