# Hackathon Synergy Agent
**Author:** Samarth Shendre
**Published:** January 16, 2026

## Overview
A RAG-powered conversational AI assistant for the Algolia Agent Studio Challenge that addresses "idea paralysis" among Indian engineering students by generating localized, tech-stack-aware hackathon project ideas from 47k indexed records.

## Key Concepts
- Algolia Agent Studio for real-time retrieval
- 20k GitHub hackathon repositories indexed
- dev.to #devchallenge posts indexed
- Indian hackathon events (Smart India Hackathon, JHUB, T-Hub)
- Geo-tagged university context with latitude/longitude
- FastAPI backend on Vercel, React Native + Expo mobile demo
