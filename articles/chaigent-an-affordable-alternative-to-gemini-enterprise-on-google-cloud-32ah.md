# Chaigent: An affordable alternative to Gemini Enterprise on Google Cloud
**Author:** Mederic Hurier
**Published:** February 6, 2026

## Overview
Cost-effective alternative to Google Gemini Enterprise using Chainlit UI on Cloud Run, Vertex AI Agent Engine, and Cloud SQL with consumption-based pricing.

## Key Concepts
- Three-layer architecture: Chainlit UI + Vertex AI Agent Engine + Cloud SQL
- Consumption-based pricing vs flat per-seat licensing ($7-35/user/month)
- Code-first approach using Python and Google ADK framework
- Target: large audiences with infrequent usage where per-seat costs are prohibitive
- Maintains modern chat features: auth, streaming, persistent history
