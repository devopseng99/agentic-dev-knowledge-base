# Claude Just Shipped Finance Agent Templates: Pitches, Valuations, and Month-End Close
**Author:** RAXXO Studios  **Published:** May 7, 2026

## Overview
Anthropic released pre-built Claude agent templates targeting finance teams. These templates provide workflow structures for pitch deck creation, valuation reviews, and month-end accounting processes. They are starting points, not finished applications.

## Key Concepts

### What Templates Actually Are
Templates combine pre-built workflows with tools, prompts, and structure. They offer "a starting point that already knows the steps" but require adaptation to specific use cases.

### Three Installation Paths
1. **Cowork Plugin** — For enterprise teams integrating workflows into existing collaboration spaces
2. **Claude Code Plugin** — For developers using terminal-based workflows
3. **Managed Agents** — Production-ready via Anthropic's hosted runtime

### Three Named Workflows

**Pitch Building**
Handles structured assembly: company background, market sizing, comparable transactions, and deal structure. Claude manages the "painful 60%" while humans refine judgment calls.

**Valuation Review**
Forensic analysis including comps review, DCF model validation, assumption flagging, and memo generation. Described as "auditable" because output is human-reviewed.

**Month-End Close**
Account reconciliation, invoice matching, variance flagging, and close package assembly — "the exact shape Claude is good at."

### Solo Studio Application
Tested on personal finance tasks:
- **Quarterly invoicing** — Good fit
- **Prepaid retainer tracking** — Mostly applicable
- **German VAT prep** — Requires significant adaptation
- **Receipt processing** — Viable with proper tool integration

### Critical Assessment
The templates follow a US accounting framework. Non-US operators should "read the cookbooks for the agent design, then build their own version." This is described as "still a 10x speedup over starting from a blank prompt."

### When Templates Work
Templates work where "the shape of the work is predictable enough that a generic template actually saves time." Success depends on matching installation path to team structure and adapting data sources accordingly.
