# AG-2 in Practice #1 - What is AG-2 and How Does It Work?
**Author:** Daniel Azevedo
**Published:** May 25, 2025

## Overview
Introduction to AG2 framework covering four core components: Agents, Orchestrators, Patterns, and Tools, plus AG2 Studio visual interface.

## Key Concepts
- Agents: Specialized characters with assigned LLMs, roles, and tool access
- Orchestrators: Control center managing communication flow and termination logic
- Patterns: Pre-built templates for linear pipelines, debates, delegation
- Tools: External functions for Python execution, HTTP, database queries
- AG2 Studio: Visual platform for no-code agent configuration
