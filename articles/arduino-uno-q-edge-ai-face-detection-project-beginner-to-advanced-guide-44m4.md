# Arduino UNO Q Edge AI & Face Detection Project (Beginner to Advanced Guide)
**Author:** David Thomas  **Published:** 2026-04-20

## Overview
Introduces the Arduino UNO Q, a next-generation development board combining microcontroller capabilities with Linux-based computing and AI functionality. Explores the board's dual-processor architecture and practical implementation through face detection demonstrations.

## Key Concepts
- Dual Processor Architecture: Separate processors handle AI/networking and real-time hardware control simultaneously
- Hybrid Operating Modes: Standalone SBC mode or connected to a laptop
- Edge AI Processing: Local face detection without cloud dependency
- Brick-Based Programming: Visual block programming via Arduino App Lab with code option
- Auto-Run Feature: Programs execute automatically at startup
- Hardware Specifications: Quad-core ARM processor (2 GHz), WiFi, Bluetooth, dedicated GPU
- Real-World Applications: Security systems, robotics, vision-based automation, IoT dashboards
- Learning Outcomes: Embedded systems, Linux development, edge AI, hardware-software integration
- Face detection running locally without cloud API calls
