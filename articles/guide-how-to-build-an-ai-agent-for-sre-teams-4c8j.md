# Guide: How to build an AI Agent for SRE Teams
**Author:** Kelsey Ellis
**Published:** November 7, 2024

## Overview
Building an AI agent for SRE teams using Chainlit, GPT-4o Assistants API, with streaming responses, document search, and PagerDuty integration.

## Key Concepts
- Chainlit for out-of-box conversational interfaces (Slack integration)
- OpenAI Assistants API with custom instructions and file search
- Real-time streaming for ChatGPT-like typing effect
- ReAct and RAG techniques for incident response
- Multi-agent broker system for diverse task types
- Function calling for external service queries (PagerDuty)
- Security: data sanitization, avoid training on sensitive info
