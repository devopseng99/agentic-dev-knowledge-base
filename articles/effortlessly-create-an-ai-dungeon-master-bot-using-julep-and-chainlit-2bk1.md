# Effortlessly Create an AI Dungeon Master Bot Using Julep and Chainlit
**Author:** Philip Balbas
**Published:** May 2, 2024

## Overview
AI Dungeon Master bot using Julep (agents, sessions, memory) and Chainlit (chat UI) with GPT-4 Turbo for dynamic TTRPG narratives.

## Key Concepts
- Agent with instructions for multiple RPG systems (D&D, Call of Cthulhu, Pathfinder)
- Session management for conversation context and player history
- Built-in memory for game state without custom database
- Async architecture for Chainlit compatibility
- Chainlit decorators: @cl.on_chat_start, @cl.on_message
