# Build Your Own ChatGPT App (Run It Locally)
**Author:** Julien Avezou
**Published:** February 4, 2026

## Overview
Build ChatGPT Apps (interactive widgets within conversations) with MCP Server backend, sandboxed iframe widgets, and local development via ngrok.

## Key Concepts
- ChatGPT Apps: interactive widgets in conversations
- Three components: MCP Server (tools/resources), Widget (HTML iframe), ChatGPT (orchestrator)
- Local development uses ngrok for HTTPS tunneling
- State management prioritizes local widget storage
