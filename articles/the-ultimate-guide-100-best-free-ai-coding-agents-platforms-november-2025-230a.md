# The Ultimate Guide: 100+ Best Free AI Coding Agents & Platforms (November 2025)
**Author:** Chirag
**Published:** November 22, 2025

## Overview
Comprehensive guide to 100+ free AI coding tools organized by category, from "God Tier" essentials to specialized tools.

## Key Concepts

### God Tier Tools
1. **Google Antigravity** -- Cloud IDE with unlimited free preview access to Gemini 3 Pro, Claude 3.5 Sonnet, GPT
2. **Amazon Kiro** -- Cloud IDE with 50 monthly credits
3. **Bolt.new** -- Web app builder with 150k-300k daily tokens
4. **Firebase Studio** -- Google's cloud-based VS Code with Gemini integration
5. **Windsurf** -- IDE with unlimited autocomplete and free chat (Llama 3.1 70B)

### Chinese Tech Platforms
Alibaba, Baidu, ByteDance, DeepSeek, Tencent, iFlytek, Zhipu AI, Moonshot -- free or heavily subsidized

### Categories Covered
- Big Tech solutions (Google AI Studio, GitHub Copilot for students/OSS, Amazon Q)
- VS Code extensions (20+ including Cline, Continue, CodeGPT)
- CLI tools (Warp, Gemini CLI, Aider)
- Web builders (Lovable, V0.dev, StackBlitz)
- Data science notebooks (DeepNote, Hex, Jupyter AI)
- Next-gen IDEs (Cursor, Zed, PearAI)

### Rotation Strategy
Combine free tools across categories for effective "unlimited" free usage by rotating between platforms as limits are reached.
