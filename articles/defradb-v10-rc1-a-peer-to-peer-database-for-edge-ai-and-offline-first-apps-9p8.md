# DefraDB v1.0 RC1: Peer-to-Peer Database for Edge AI and Offline-First Apps
**Author:** addobot  **Published:** 2026-03-18

## Overview
Announces DefraDB v1.0 RC1, a peer-to-peer database designed for edge AI applications and offline-first deployments. Enables distributed robot fleets and edge devices to share data without centralized server infrastructure.

## Key Concepts
- Peer-to-peer data synchronization for edge devices
- Offline-first operation: works without network connectivity
- Conflict-free replicated data types (CRDTs) for distributed consistency
- GraphQL API for data queries
- Privacy-preserving data sharing between robots
- Embedded database suitable for resource-constrained devices
- Decentralized AI training data collection
- Use cases: multi-robot coordination, drone swarms, IoT sensor networks
- Integration patterns with robot middleware (ROS2, custom stacks)
