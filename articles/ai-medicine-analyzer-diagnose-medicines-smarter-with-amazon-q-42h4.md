# AI Medicine Analyzer – Diagnose Medicines Smarter with Amazon Q
**Author:** Er Sumit  **Published:** May 6, 2025

## Overview
An intelligent web application enabling users to analyze medications through AI-powered insights. Provides structured information about medicine uses, dosages, side effects, precautions, and drug interactions in an accessible chat-like interface. Addresses the challenge of patients struggling to comprehend medication instructions. Built for the Amazon Q Developer "Quack The Code" Challenge.

## Key Concepts
- Natural language medication analysis with structured breakdowns
- Amazon Q used for code generation, component refactoring, and API route setup
- Chat-like interface for medication queries
- GitHub: https://github.com/sumitKcs/medicine_analyzer
- Live Demo: https://medilyze.netlify.app
- Future: streaming AI responses, drug interaction detection, AWS Bedrock integration, HIPAA-compliant enterprise version, medical image analysis
