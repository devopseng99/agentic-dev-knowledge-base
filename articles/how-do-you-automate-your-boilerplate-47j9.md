# How Do You Automate Your Boilerplate?
**Author:** Marc Nevin  **Published:** February 24, 2020

## Overview
Marc's team sought to streamline project initialization by automating repetitive boilerplate setup. They explored GitHub templates, GitHub Actions automation, and language-specific tools for code generation.

## Key Concepts
- **GitHub Templates**: Repository templates enabling quick project creation without manual duplication
- **GitHub Actions**: CI/CD automation integrated into repositories
- **Cookiecutter**: Python tool for scaffolding projects with templated files
- **Hygen.io**: Code generation framework supporting variable substitution and post-generation scripts
- **Degit**: Repository cloning tool enabling template reuse from any GitHub repo
- **Dependency-based abstraction**: Packaging common functionality as reusable dependencies (traits/mixins)
