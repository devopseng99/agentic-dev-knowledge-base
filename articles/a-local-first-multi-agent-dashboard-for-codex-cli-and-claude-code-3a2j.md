# A Local-First Multi-Agent Dashboard for Codex CLI and Claude Code
**Author:** volition79
**Published:** April 19, 2026

## Overview
Sonol Multi Agent is a local-first orchestration system for Codex CLI and Claude Code, converting single requests into structured multi-agent plans with dashboard review and real-time monitoring.

## Key Concepts
- Converts requests into structured multi-agent plans
- Dashboard review and approval before execution
- Real-time monitoring of sub-agent flow
- Two components: sonol-multi-agent (planning/orchestration) and sonol-agent-runtime (dashboard/execution)
- Local SQLite database as authoritative source of truth
- Tested on Codex CLI (Windows WSL), Claude Code (Windows, macOS)
