# Build a Production-Ready, Intelligent Chatbot With the MongoDB Chatbot Framework
**Author:** Ben Perlmutter
**Published:** January 26, 2024

## Overview
MongoDB's TypeScript framework for production-grade chatbots with three core modules: Chatbot Server (Express.js with Atlas Vector Search for RAG), Chatbot UI (React components), and Ingest CLI (data ingestion for embeddings).

## Key Concepts
- Chatbot Server: TypeScript/Express.js with Atlas Vector Search for RAG
- Chatbot UI: Customizable React components with suggested prompts and modal views
- Ingest CLI: Pulls data, chunks documents, creates embeddings, stores in MongoDB
- Full TypeScript stack for type-safe custom logic
- Built-in data persistence and security features
