# Amazon Managed Streaming for Apache Kafka (MSK)
**Author:** AWS Builders
**Published:** 2024

## Overview
Guide to Amazon MSK -- fully managed Apache Kafka service that handles provisioning, configuration, and maintenance of Kafka clusters.

## Key Concepts
- Managed Kafka eliminates operational overhead of running clusters
- Automatic broker patching and cluster scaling
- Integration with AWS services (Lambda, Kinesis, S3)
- VPC-based networking for security
- MSK Connect for managed Kafka Connect deployments
- Monitoring via CloudWatch metrics
