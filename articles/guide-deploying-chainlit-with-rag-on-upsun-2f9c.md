# Deploying Chainlit with RAG on Upsun
**Author:** Celeste van der Watt
**Published:** February 12, 2025

## Overview
Deploy Chainlit applications with RAG on Upsun platform, covering two implementations: OpenAI Assistants (built-in file handling) and llama_index (customizable local vector storage).

## Key Concepts
- Chainlit framework for AI applications
- Two RAG implementations: OpenAI Assistants and llama_index
- SQLite-based authentication
- Cloud deployment and scalability on Upsun
