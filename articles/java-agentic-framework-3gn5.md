# Java Agentic Framework
**Author:** vishalmysore
**Published:** January 25, 2026

## Overview
Tools4AI: annotation-driven Java framework for AI agents. @Agent marks classes, @Action defines executable methods, @Prompt handles field processing. Supports HTTP actions via Swagger, shell scripts, image processing, Selenium integration, and Spring framework.

## Key Concepts
- @Agent: class-level annotation, all @Action methods added to prediction list
- @Action: method-level with descriptions for AI prompt mapping
- @Prompt: field-level for translation, date formatting, field ignoring
- Multi-action workflows and script processing for autonomous sequences
- High-risk actions with configurable risk levels
