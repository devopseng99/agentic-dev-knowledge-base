# Top 18 Open Source AI Agent Projects with the Most GitHub Stars
**Author:** NocoBase
**Published:** August 15, 2025

## Overview
Catalogs top open-source agent projects: Dify (110k stars), Lobe Chat (64.3k), RAGFlow (62.1k), OpenHands (62k), MetaGPT (57.8k), Huginn (47k), Unsloth (43.8k), Mem0 (38.1k), AgentGPT (34.7k), Khoj (31.3k), and more. Key principle: agents excel at exploration but struggle with strict rule-following.

## Key Concepts
- Agents work best in unstable processes with human oversight
- Good for: data science, creative writing, code refactoring
- Caution: mission-critical enterprise automation, high-stakes decisions
- Includes Java diagnostic tool Arthas (36.6k stars)
