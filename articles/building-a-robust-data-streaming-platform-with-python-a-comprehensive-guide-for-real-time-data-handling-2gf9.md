# Building a Robust Data Streaming Platform with Python
**Author:** Amit Chandra
**Published:** September 22, 2024

## Overview
Production-ready data streaming systems using Python and Apache Kafka. Covers Lambda/Kappa architectures, Docker-based Kafka, Apache Avro schemas, stream processing, and security.

## Key Concepts
- Lambda architecture (batch + stream) vs Kappa architecture (stream-only)
- Docker Compose for Kafka + Zookeeper deployment
- Apache Avro with schema registry for data serialization
- Complex event detection for fraud prevention
- TLS encryption and SASL authentication for security
- Monitoring with Prometheus/Grafana
- Backpressure handling with rate limiting
- Storage: TimescaleDB, InfluxDB, PostgreSQL
