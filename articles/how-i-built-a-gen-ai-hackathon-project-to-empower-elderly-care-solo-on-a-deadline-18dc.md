# How I Built a Gen AI Hackathon Project to Empower Elderly Care
**Author:** ByteMotive Blog
**Published:** April 14, 2025

## Overview
GuardianAI is a lightweight multi-agent system for elderly care built during "Hack the Future: A GenAI Sprint" Hackathon. Three specialized agents handle health monitoring, reminders, and emergency alerts.

## Key Concepts
- Health Monitor Agent: processes vital signs and identifies anomalies
- Reminder Agent: personalized daily notifications for medication and hydration
- Alert Agent: aggregates and displays emergency warnings
- Python 3.11, pandas, CLI interface
