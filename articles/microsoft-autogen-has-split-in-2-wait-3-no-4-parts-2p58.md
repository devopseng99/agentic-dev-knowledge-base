# Microsoft Autogen Has Split in 2... Wait 3... No, 4 Parts
**Author:** Maxim Saplin
**Published:** November 18, 2024

## Overview
Analysis of AutoGen framework fragmentation into four competing versions: AutoGen 0.4 (new event-driven architecture), AutoGen 0.2 (legacy), AG2 (community fork), and Semantic Kernel integration plans.

## Key Concepts
- AutoGen 0.4: Complete rewrite with actor model, `autogen-agentchat` and `autogen-core` packages
- AutoGen 0.2: Legacy synchronous version, backward compatible
- AG2: Community fork maintaining 0.2 compatibility, available as `ag2`, `autogen`, `pyautogen`
- Semantic Kernel: Microsoft plans merging 0.4 runtime into enterprise SK framework
- Five competing PyPI packages creating significant developer confusion
