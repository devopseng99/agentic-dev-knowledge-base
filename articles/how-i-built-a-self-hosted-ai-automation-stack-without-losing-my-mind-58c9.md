# How I Built a Self-Hosted AI Automation Stack Without Losing My Mind
**Author:** devtips
**Published:** August 28, 2025

## Overview
Building a developer-friendly automation system using n8n, Docker, and open-source AI tools to replace expensive no-code platform subscriptions.

## Key Concepts
- n8n for self-hosted workflow orchestration
- Docker containerization for deployment
- MCP (Mass Code Prompting) CLI for scaling LLM prompts
- AI models via OpenRouter, Ollama, or LM Studio
- Workflow: file drop -> AI processing via MCP -> output routing (Slack, email, GitHub)
- Zero monthly subscription costs with unlimited execution
- Tools: LM Studio, OpenRouter, Ollama, Watchtower, Portainer
