# How to Build a Full-Stack App with an AI Coding Agent
**Author:** Nick Peterson
**Published:** December 12, 2025

## Overview
Hybrid workflow for 2026 development using AI coding agents. Covers the agentic tech stack, master prompts, build loops, and the critical "final 20%" where humans are essential.

## Key Concepts

### Phase 1 - Agentic Tech Stack
- Orchestrator: Windsurf or Cursor (IDEs with agent capabilities)
- Backend: Supabase or Convex (serverless)
- Frontend: React (Vite) or Next.js

### Phase 2 - Master Prompt
Create a `specs.md` file with detailed requirements. Instruct the agent to plan before coding.

### Phase 3 - Build Loop
Agent scaffolds the project, builds features iteratively, self-heals when errors occur.

### Phase 4 - Reality Check (Final 20%)
"AI agents are incredible at boilerplate and standard patterns" but struggle with complex business logic, security, and high-scale architecture. Professional developers should audit and refactor for production.

### Phase 5 - Deployment
Use agents to generate CI/CD configurations (GitHub Actions, Vercel deployment).
