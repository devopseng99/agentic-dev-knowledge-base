# Smart Supply AI-Agent Development Kit Hackathon with Google Cloud
**Author:** Mudigonda Venkata Gopi Jayaram
**Published:** June 22, 2025

## Overview
A multi-agent logistics intelligence platform built using Google's Agent Development Kit (Python) for the ADK Hackathon. Sequential agent pipeline with four specialized agents for supply chain automation.

## Key Concepts
- Fetch Agent (data retrieval)
- Risk Agent (route/product risk assessment)
- Optimization Agent (cost and delay reduction)
- Report Agent (structured insights generation)
- Domain-specific logistics heuristics for cold-chain pharma and cross-border routing

### Setup
```bash
git clone https://github.com/jayaram9196/DemoAgent.git
cd DemoAgent
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
adk web
```
