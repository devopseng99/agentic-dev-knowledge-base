# Build Your Own ChatGPT for $100: Introducing nanochat, the Hackable LLM Blueprint
**Author:** GitHubOpenSource
**Published:** October 18, 2025

## Overview
Nanochat is a complete end-to-end LLM implementation covering tokenization through web serving. Trainable for approximately $100 via provided automation scripts. Includes a ChatGPT-like web UI for inference.

## Key Concepts
- Complete pipeline: tokenization, pretraining, fine-tuning, evaluation, inference, and serving
- Educational resource emphasizing transparency and customization
- Train on 8XH100 nodes within approximately four hours via speedrun.sh script
- Stars: 25,311 | Forks: 2,530
- Tech Stack: Python
