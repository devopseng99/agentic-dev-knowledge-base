# Building a Copilot Code-First with Azure AI

**Author:** Nitya Narasimhan, Ph.D
**Published:** March 7, 2024

## Overview
Explores building generative AI applications using code-first methodology with Azure AI Studio. Covers the LLM Ops paradigm shift from ML Ops and the three development phases: ideation, augmentation, operationalization.

## Key Concepts

### LLM Ops vs ML Ops
Previous AI focused on custom models trained on finite datasets for predictions. Modern focus: prompt engineering and fine-tuning pre-trained models for content generation.

### Three Development Phases
1. **Ideation:** Explore models, engineer prompts
2. **Augmentation:** Build and manage AI projects, assess quality
3. **Operationalization:** Deploy and monitor solutions

### Azure AI Studio
Unified platform supporting the entire workflow with two paths: browser-based UI for low-code or CLI/SDK for professional developers.
