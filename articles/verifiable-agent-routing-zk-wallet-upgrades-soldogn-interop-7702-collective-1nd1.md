# Verifiable Agent Routing, ZK Wallet Upgrades, Soldogn Interop, 7702 Collective

**Author:** Alexandra (Etherspot)
**Published:** May 7, 2026

## Overview

Weekly digest covering four major Ethereum account and chain abstraction developments.

## Soldogn Interop and Glamsterdam

Over 100 Ethereum contributors gathered in Svalbard. Key outcomes: post-upgrade gas limit floor of 200 million, advancing external PBS implementations, addressing block construction, state growth, and gas accounting.

## ZK-Based Wallet Upgrades

New proposal using EIP-7702 delegation, zero-knowledge proofs, and hidden public keys to upgrade existing wallets to post-quantum security without asset migration. Protects against quantum key extraction.

## Autonomous State Gateway (ASG)

New proposal standardizing how off-chain solver outputs route to verifiable on-chain states. Treats results as Proposed State Transitions with commitment, collateral reservation, verification, dispute windows, and settlement.

## 7702 Collective Launch

Particle Network and eight co-founding organizations announced this coalition for infrastructure-driven crypto adoption, releasing "The State of Crypto Innovation" inaugural report.
