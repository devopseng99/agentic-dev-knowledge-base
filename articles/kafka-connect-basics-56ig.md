# Kafka Connect Basics
**Author:** nuculabs_dev
**Published:** 2024

## Overview
Introduction to Kafka Connect as a tool for streaming data between Kafka and other systems. Distributed and scalable by default with extensive connector ecosystem.

## Key Concepts
- Distributed streaming between Kafka and external systems
- Pre-built connectors for databases, file systems, cloud services
- Automatic scaling and fault tolerance
- Configuration-driven setup (no custom code for standard connectors)
- Integration point for AI/ML pipelines connecting to data stores
