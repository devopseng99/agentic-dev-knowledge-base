# Lightning-Fast Code Assistant with Groq in VSCode

**Author:** Daniel (San) Avila
**Published:** July 21, 2024

## Overview
Tutorial for integrating Groq's cloud-based language models as a code assistant in Visual Studio Code using the CodeGPT extension, leveraging Groq's LPU Inference Engine for exceptional compute speed.

## Key Concepts

### What is Groq?
Groq is the AI infrastructure company that delivers fast AI inference through its LPU (Language Processing Unit) Inference Engine, providing exceptional compute speed, quality, and energy efficiency.

### Setup Instructions

**Step 1: Create Groq Account**
- Visit https://console.groq.com/
- Navigate to the API Keys section
- Create and copy a new API Key

**Step 2: Install CodeGPT Extension**
- Open VSCode
- Install the CodeGPT extension

**Step 3: Connect to Groq**
- Open CodeGPT in VSCode
- Select Groq as the provider
- Click "Connect"
- Paste your API Key and confirm connection

**Step 4: Select Model**
- Choose from available models (llama3-70b-8192 recommended)
- CodeGPT maintains an updated model list

### Key Features
- **Explain a File:** Select /Explain option, use @ to select project files, get comprehensive explanations
- **Document a File:** Automatically generate documentation for code files
