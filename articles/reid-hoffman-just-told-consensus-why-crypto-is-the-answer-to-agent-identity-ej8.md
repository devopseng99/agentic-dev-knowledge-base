# Reid Hoffman Just Told Consensus Why Crypto Is the Answer to Agent Identity

**Author:** Aaron Schnieder
**Published:** May 7, 2026

## Overview

At Consensus Miami, LinkedIn co-founder Reid Hoffman claimed cryptocurrency provides "the obvious answer" to agent identity on the open internet. Enterprise identity solutions already exist (Google SPIFFE, Microsoft Entra Agent ID, Okta), but the open internet -- where agents transact with strangers -- still lacks a trust layer.

## Enterprise vs Open Internet

Google Cloud designated agent identity as a first-class IAM principal type. Microsoft introduced Entra Agent ID within Agent 365 ($15/user). Okta launched AI agent identity management. These use SPIFFE standards with cryptographic protection.

Hoffman's focus: public internet scenarios where agents conduct transactions with unfamiliar agents beyond corporate boundaries.

## The Three Shipped Layers

1. **Identity** -- Google SPIFFE, ERC-8004, FIDO, Microsoft Entra
2. **Payments** -- x402, Stripe MPP, Visa Intelligent Commerce
3. **Governance** -- Agent 365, Forrester AEGIS, Gartner Guardian

## The Missing Layer: Reputation

While identity answers "who," payments enable transactions, and governance controls capabilities, reputation reveals actual performance histories. Key stats:
- 72% of organizations operate AI agents in production
- 92% cannot safely scale them
- 66% grant agents equal or greater access than humans
- Only 17% employ designated security leaders for agent governance

## Key Developments

Experian launched Agent Trust with "Human-to-Agent Binding." Gartner published its inaugural Market Guide for Guardian Agents. The identity dark matter problem: approximately half of enterprise identity activity occurs outside centralized IAM systems.

On-chain reputation computed from genuine escrowed transactions represents infrastructure needed for agents to transact with strangers across open networks.
