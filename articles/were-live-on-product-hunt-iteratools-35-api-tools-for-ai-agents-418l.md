# We're live on Product Hunt: IteraTools, 35+ API tools for AI agents
**Author:** Fred Santos  **Published:** 2026-03-08

## Overview
IteraTools: a unified REST API consolidating 35+ tools into a single platform with one authentication key, solving the problem of AI agent developers integrating multiple SDKs and API keys for different capabilities.

## Key Concepts

### The Problem
AI agent developers must integrate multiple SDKs and API keys for: image generation, audio transcription, web scraping, code execution, and messaging — each requiring separate tools and error handling.

### The Solution
One platform, one auth key, consistent pricing. Pay-per-use model with costs like $0.003 for image generation and $0.002 for web scraping.

### 35+ Tools Across 8 Categories

- **Image tools:** generation, background removal, resizing, OCR
- **Video capabilities:** generation, frame extraction, screenshots
- **Web tools:** scraping, extraction, PDF handling, browser automation
- **Audio:** text-to-speech (100+ voices), transcription
- **Messaging:** WhatsApp integration with conversation tracking
- **Code execution:** Python, JavaScript, Bash via E2B sandboxes
- **Data tools:** charts, spreadsheets, DNS, WHOIS lookups
- **Utilities:** QR codes, weather, currency conversion, translation

### MCP Support
All tools work as Model Context Protocol servers, compatible with Claude Desktop, Cursor, and Claude Code.

### Pricing Model
- USDC-based payment on Base blockchain
- No subscriptions
- $1 minimum funding enables hundreds of API calls
