# AI Agents In Java
**Author:** vishalmysore
**Published:** September 15, 2024

## Overview
Discusses multi-agent AI systems in Java with IEEE and FIPA standards. Covers Agent Communication Language (ACL) with command types: INFORM, REQUEST, QUERY_IF, PROPOSE, AGREE, REFUSE, FAILURE. Practical example: Gemini agent sends questions to OpenAI agents, scores responses.

## Key Concepts
- IEEE P7001: transparency in autonomous systems
- IEEE P2874: standardized communication protocols for AI agents
- FIPA: Foundation for Intelligent Physical Agents standards
- ACL message types for structured agent interaction
- Multi-agent scoring and best-answer selection
