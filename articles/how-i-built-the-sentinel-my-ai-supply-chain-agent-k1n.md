# How I Built The Sentinel: My AI Supply Chain Agent
**Author:** Felicia Ebikon
**Published:** January 26, 2026

## Overview
A 48-hour hackathon project creating an intelligent inventory monitoring system. The Sentinel automatically analyzes stock levels and alerts users when restocking is needed, using Google Gemini as the analytical engine, Kestra for workflow orchestration, Next.js for the UI, and Vercel for deployment.

## Key Concepts
The primary challenge involved formatting Gemini's responses for Kestra to parse. Restructuring prompts to return JSON enabled proper data flow between components. Combines workflow automation tools with AI for production-ready solutions.
