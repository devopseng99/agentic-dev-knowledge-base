# Apache Gravitino 1.0.0
**Author:** Alex Yan (Datastrato)
**Published:** October 5, 2025

## Overview
Apache Gravitino evolves from metadata catalog to "smart catalog" with automated actions, MCP server integration for AI assistants, and enhanced governance.

## Key Concepts
- Statistics, Policy, and Job Systems for automated table compaction and TTL management
- MCP Server Integration enables natural language interaction with metadata via Claude/Cursor
- Extended support for Apache Iceberg 1.9.0, Paimon 1.2.0, Kafka metadata
- Enhanced governance: enforced access control + RBAC
- Multiple storage locations in model management
- Requires JDK 17
