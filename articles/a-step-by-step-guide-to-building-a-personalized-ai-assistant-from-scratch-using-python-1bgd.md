**Author:** DevOps VN
**Published:** June 24, 2025

## Overview
Build a custom AI "second brain" using three components: the Brain (LLM), Memory (vector database), and Connectors (Python code). RAG pipeline implementation.

- Three components: Brain (LLM), Memory (vector DB), Connectors (Python)
- Retrieval-Augmented Generation (RAG) pipeline
- 5 parts, 9 chapters: from basics through web interface to privacy/security
- No AI expertise required, basic Python sufficient
