# I Won The Build4Students Hackathon! WeRelaySyllabus Devlog
**Author:** Hamd Waseem
**Published:** February 28, 2026

## Overview
14-year-old developer won first place in the Build4Students Hackathon (708 participants, judges from Meta, Anthropic, Capital One) with WeRelaySyllabus, a platform to crowdsource course syllabus calendars.

## Key Concepts
- Firebase Firestore for real-time capabilities (chosen over AWS Lambda/DynamoDB)
- ICS calendar file generation with VCALENDAR/VEVENT blocks
- Neo-brutalism styling (bold black shadows, uppercase typography)
- Daylight Savings offset correction, dynamic end-time calculation
- Deployed on Vercel
